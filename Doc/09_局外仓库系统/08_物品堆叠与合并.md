# 09-08 物品堆叠与合并

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

入库和整理时自动合并可堆叠物品，减少仓库空间浪费。

## 2. 自动堆叠条件

- item_id 相同。
- quality 相同。
- 未达到 stack_limit。
- 特殊状态一致，V0.1 可忽略复杂状态。

## 3. 堆叠限制

stack_limit 来自 07 ItemDefinition。

合并流程：

```text
先合并已有堆叠
-> 剩余数量创建新堆叠
-> 新堆叠找空格
-> 返回无法放入数量
```

## 4. AI/MCP 开工提示词

```text
请实现仓库物品自动堆叠与合并。
要求：
1. 入库时优先合并 item_id 和 quality 相同的未满堆叠。
2. stack_limit 从 07 ItemDefinition 读取。
3. 合并后还有剩余时再占用新格。
4. 空间不足返回剩余数量，不得吞物。
5. 排序/整理后物品总数必须一致。
```

## 5. 验收标准

- 相同物品能合并。
- 不同品质不合并。
- 超出堆叠上限开新堆。
- 空间不足剩余明确。

## 6. 暂不实现

- 不同品质自动升降级。
- 手动拆分堆叠 UI。
