## 1. 拆分目标

09 模块负责局外长期物品保存，是 08 撤离成功后的收益承接点，也是研究所、制作所和出发准备的查询来源。

落地闭环：

```text
结算物品 -> 手动/一键入库 -> 仓库占格/堆叠 -> 满溢处理 -> 保存 -> 研究/制作/出发查询
局外仓库物品 -> 商人选择出售数量 -> 扣除仓库物品 -> 增加玩家货币 -> 保存
```

## 3. 施工硬约束

- 不要静默替玩家丢东西。
- 初期仓库不要太小，避免立刻卡住结算。
- 仓库不计算负重。
- 入库失败必须保留结算条目。
- 研究所/制作所只能通过查询接口读取或请求消耗，不直接改仓库数组。
- 商人出售必须通过 WarehouseManager 扣物和 CurrencyWallet 加币，不允许 UI 直接删仓库物品或改货币数。
- 货币以 currency_id 存档，V0.1 只有 mine_coin，后续可扩展多货币。
- 局外仓库、商人、研究所、制作所中凡是道具图标，都必须接入 `18_局外道具详情Tooltip.md`；该 Tooltip 只在局外 BaseScene 启用，不直接接入局内背包、容器、拾取提示或结算界面。
