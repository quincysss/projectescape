# 09-04 手动入库、一键入库与优先级

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

为 08 结算界面提供手动入库和一键入库策略。

## 2. 手动入库

手动入库：

- 玩家选择条目。
- 可选择数量。
- 预览仓库空间。
- 确认后尝试入库。

## 3. 一键入库

一键入库优先级建议：

```text
任务/关键材料
-> 高品质物品
-> 可堆叠材料
-> 普通资源
-> 低价值杂物
```

如果仓库空间不足，优先保留高价值物品入库。

## 4. AI/MCP 开工提示词

```text
请实现局外仓库手动入库和一键入库策略。
要求：
1. 手动入库支持选择物品和数量。
2. 一键入库按任务/关键材料、高品质、可堆叠材料、普通资源、低价值杂物排序。
3. 仓库空间不足时返回未入库条目。
4. 不要静默丢弃未入库物品。
5. 入库结果要能让 08 更新结算条目状态。
```

## 5. 验收标准

- 手动入库可控。
- 一键入库优先级稳定。
- 满仓时不会吞物。

## 6. 暂不实现

- 玩家自定义优先级。
- 自动分解低价值物。
