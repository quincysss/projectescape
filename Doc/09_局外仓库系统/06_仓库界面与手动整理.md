# 09-06 仓库界面与手动整理

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现局外仓库界面，让玩家查看长期资源、手动整理、丢弃，并为制造所、研究所、白天货台和夜晚出发准备提供物品来源。

仓库入口位于白天准备阶段功能入口中，与制造所、研究所、图鉴平级。局外不再使用“顶部页签 + 商人页签”的旧结构。

## 2. 必须显示

- 仓库网格。
- 物品图标。
- 物品数量。
- 品质颜色。
- 已用容量/总容量。
- 选中物品详情。
- 当前持有矿币数量，或全局顶部货币显示。
- 道具图标 hover 详情 Tooltip，规则见 `18_局外道具详情Tooltip.md`。

## 3. 建议显示

- 类型筛选。
- 品质筛选。
- 搜索。
- 排序按钮。

## 4. 手动整理

支持：

- 移动物品位置。
- 合并堆叠。
- 排序。
- 丢弃。

所有操作必须走 WarehouseManager。

仓库界面不提供旧式直接出售。玩家获得货币的主路径是：制造所把原材料加工为 `sale_good` 可售物资，白天开店时把可售物资摆上货台并完成成交。

## 5. AI/MCP 开工提示词

```text
请实现局外仓库界面和手动整理。
要求：
1. 显示仓库网格、物品图标、数量、品质、容量、选中详情。
2. 支持移动物品、合并堆叠、排序、丢弃入口。
3. UI 不直接改仓库数组，必须调用 WarehouseManager。
4. 操作失败显示 reason。
5. 仓库界面可从结算溢出处理返回后继续入库。
6. 白天准备阶段功能入口为：仓库、制造所、研究所、图鉴；夜晚阶段不显示仓库入口。
7. 仓库物品图标必须接入局外道具详情 Tooltip；hover 0.15 秒后显示图标、名称、用途标签和描述，拖拽、滚动、切换功能入口时隐藏。
```

## 6. 验收标准

- 玩家能查看仓库物品。
- 移动/合并不会丢物。
- 容量显示正确。
- 白天准备阶段可进入仓库；开店、结算、夜晚、夜间计划和出发准备阶段按 `30_昼夜局外流程_店铺营业与夜间出发准备.md` 控制入口显示。
- 仓库内有物品的图标 hover 后能显示局外道具详情 Tooltip，空格子不显示。
- 图鉴入口能进入 `19_局外道具图鉴系统.md` 定义的四列卡片界面。

## 7. 暂不实现

- 高级搜索。
- 仓库内部多标签页。
