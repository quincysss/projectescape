# 09-06 仓库界面与手动整理

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现局外仓库界面，让玩家查看长期资源、手动整理、丢弃和为出发准备取物。

## 2. 必须显示

- 仓库网格。
- 物品图标。
- 物品数量。
- 品质颜色。
- 已用容量/总容量。
- 选中物品详情。

## 3. 建议显示

- 类型筛选。
- 品质筛选。
- 搜索。
- 排序按钮。

## 4. 手动整理

支持：

- 移动物品位置。
- 合并堆叠。
- 排序。
- 丢弃。

所有操作必须走 WarehouseManager。

## 5. AI/MCP 开工提示词

```text
请实现局外仓库界面和手动整理。
要求：
1. 显示仓库网格、物品图标、数量、品质、容量、选中详情。
2. 支持移动物品、合并堆叠、排序、丢弃入口。
3. UI 不直接改仓库数组，必须调用 WarehouseManager。
4. 操作失败显示 reason。
5. 仓库界面可从结算溢出处理返回后继续入库。
```

## 6. 验收标准

- 玩家能查看仓库物品。
- 移动/合并不会丢物。
- 容量显示正确。

## 7. 暂不实现

- 高级搜索。
- 多标签页仓库。
