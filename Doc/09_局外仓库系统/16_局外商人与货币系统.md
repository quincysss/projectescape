# 09-16 旧局外商人与货币系统（已废弃）

> 状态：已废弃，不再作为实现依据。
> 废弃原因：最新局外循环已改为“白天制造可售物资 -> 开店营业 -> 货台成交获得货币 -> 夜晚出发”。旧的独立商人出售界面会绕开制造所和店铺经营主循环。
> 替代文档：`Doc/29_局外杂货店经营与界面重构路线图.md`、`Doc/30_昼夜局外流程_店铺营业与夜间出发准备.md`、`Doc/31_掉落物资与制造经济规划.md`。

## 1. 禁止继续实现的旧规则

- 不实现独立局外商人入口。
- 不实现仓库物品直接出售换矿币。
- 不实现独立出售确认面板。
- 不实现商人解锁门禁。
- 不实现 MerchantPanel 作为局外一级界面。
- 不允许 AI 继续以本文旧标题推导新代码。

## 2. 新货币来源

货币来自白天店铺营业：

```text
局内带回原材料
-> 制造所合成 sale_good 可售物资
-> 白天准备查看需求榜
-> 开店阶段摆上货台
-> 难民需求匹配后成交
-> 营业结算增加货币
```

仓库系统只提供货台上架、撤下、成交扣货和未售出回仓接口，不直接把仓库物品出售为货币。

## 3. 新货币消耗

货币消耗口见 `31_掉落物资与制造经济规划.md`：

- 地点远行费用。
- 装备和消耗品制造工时费。
- 后勤订购标准件。
- 装备维修。
- 货台扩建。
- 经营情报。
- 长期研究和设施升级。
