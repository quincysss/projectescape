# 09-17 旧随机库存与购买规则（已废弃）

> 状态：已废弃，不再作为实现依据。
> 废弃原因：最新经济中不再通过独立随机商店补材料或购买成品。装备、消耗品和高级制造所需的货币消耗，改由制造所内的“后勤订购标准件”和夜晚地点远行费用承接。
> 替代文档：`Doc/31_掉落物资与制造经济规划.md`。

## 1. 禁止继续实现的旧规则

- 不实现 `shop_stock.tab` 作为局外随机库存。
- 不实现独立购买列表。
- 不实现按商店等级刷新库存。
- 不通过购买成品装备或成品消耗品绕过制造所。

## 2. 新替代机制

如果配方需要非局内掉落的标准件，使用后勤订购：

```text
制造所配方
-> 检查局内材料
-> 检查标准件
-> 标准件不足时，可在制造所内后勤订购
-> 消耗货币获得标准件
-> 继续制造装备、消耗品或高级设施材料
```

标准件示例见 `31_掉落物资与制造经济规划.md`：

- 标准紧固件盒。
- 药剂基底。
- 标准电力接口。
- 滤芯外壳。
- 防护内衬。
