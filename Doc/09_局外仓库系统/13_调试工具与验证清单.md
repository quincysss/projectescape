# 09-13 调试工具与验证清单

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 最新口径：调试工具必须覆盖仓库、制造消耗、货台成交、货币保存和图鉴点亮；不再提供旧独立售卖调试按钮。

## 1. Debug 字段

| 字段 | 用途 |
| --- | --- |
| used_slots | 已用格 |
| total_slots | 总格 |
| item_stack_count | 仓库物品堆数 |
| total_item_count | 仓库物品总数 |
| shelf_slot_count | 当前货台格数 |
| shelf_used_count | 已上架货台格数 |
| revision | 仓库版本 |
| last_save_result | 最近保存结果 |
| currencies | 玩家持有货币字典 |
| mine_coin | 当前矿币数量 |
| collected_item_ids | 图鉴已点亮 item_id 字典 |

## 3. 验证清单

- [ ] 结算物可以入库。
- [ ] 仓库满时不吞物，剩余物返回结算界面。
- [ ] 整理后可以继续入库。
- [ ] 同物品同品质能堆叠。
- [ ] 丢弃需要确认。
- [ ] 研究/制造消耗不足时不扣物。
- [ ] 制造成功会扣除材料并生成可售物资。
- [ ] 可售物资可以上架到空货台。
- [ ] 货台初始只有 3 格。
- [ ] 货台已满时不能继续上架。
- [ ] 货台成交成功会扣除上架物资并增加 `mine_coin`。
- [ ] 货台成交失败不扣物、不加币。
- [ ] 重启后 `mine_coin` 数量保留。
- [ ] 保存后重启仍保留仓库、货台和图鉴状态。
- [ ] 图鉴默认展示所有可展示道具。
- [ ] 图鉴每行 4 张卡片，横向不滑动，纵向滚动。
- [ ] 获得过的道具点亮并保存到 `collected_item_ids`。
- [ ] 制造、成交、消耗、升级或丢弃道具后，图鉴点亮状态不消失。

## 5. 验收标准

- Debug 可复现满仓、接口消耗、制造产出和货台成交。
- Debug 可复现货币保存和消费。
- 保存/重载可验证。
- Debug 可复现图鉴点亮、清空、保存重载，以及成交/消耗后不熄灭。
