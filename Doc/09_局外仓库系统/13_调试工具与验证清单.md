# 09-13 调试工具与验证清单

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. Debug 字段

| 字段 | 用途 |
| --- | --- |
| used_slots | 已用格 |
| total_slots | 总格 |
| item_count | 物品堆数量 |
| total_item_count | 物品总数量 |
| revision | 仓库版本 |
| last_save_result | 最近保存结果 |
| currencies | 玩家持有货币字典 |
| mine_coin | 当前矿币数量 |

## 2. Debug 按钮

- Add Test Items
- Fill Warehouse
- Clear Warehouse
- Try Settlement Transfer
- Force Overflow
- Sort Warehouse
- Simulate Research Consume
- Simulate Crafting Consume
- Simulate Merchant Sell
- Add Mine Coins
- Save Warehouse
- Reload Warehouse

## 3. 验证清单

- [ ] 结算物可以入库。
- [ ] 仓库满时不吞物。
- [ ] 整理后可以继续入库。
- [ ] 同物品同品质能堆叠。
- [ ] 丢弃需要确认。
- [ ] 研究/制作消耗不足时不扣物。
- [ ] 商人出售可选择数量。
- [ ] 出售成功扣除仓库物品并增加 mine_coin。
- [ ] 出售失败不扣物、不加币。
- [ ] 重启后 mine_coin 数量保留。
- [ ] 保存后重启仍保留。

## 4. AI/MCP 开工提示词

```text
请实现 09 仓库 Debug 面板。
要求：
1. 显示已用格、总格、物品堆数、物品总数、revision、保存结果、currencies、mine_coin。
2. 提供添加测试物品、填满仓库、结算入库、强制满溢、排序、模拟研究消耗、模拟制作消耗、模拟商人出售、添加矿币、保存和重载。
3. 跑完入库、仓库、接口、保存验证清单。
```

## 5. 验收标准

- Debug 可复现满仓和接口消耗。
- Debug 可复现商人出售和货币保存。
- 保存/重载可验证。
