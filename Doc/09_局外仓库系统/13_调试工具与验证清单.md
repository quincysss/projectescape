## 1. Debug 字段

| 字段 | 用途 |
| --- | --- |
| used_slots | 已用格 |
| total_slots | 总格 |
| item_count | 物品堆数量 |
| total_item_count | 物品总数量 |
| revision | 仓库版本 |
| last_save_result | 最近保存结果 |
| currencies | 玩家持有货币字典 |
| mine_coin | 当前矿币数量 |
| collected_item_ids | 图鉴已点亮 item_id 字典 |

## 3. 验证清单

- [ ] 结算物可以入库。
- [ ] 仓库满时不吞物。
- [ ] 整理后可以继续入库。
- [ ] 同物品同品质能堆叠。
- [ ] 丢弃需要确认。
- [ ] 研究/制作消耗不足时不扣物。
- [ ] 商人出售可选择数量。
- [ ] 出售成功扣除仓库物品并增加 mine_coin。
- [ ] 出售失败不扣物、不加币。
- [ ] 重启后 mine_coin 数量保留。
- [ ] 保存后重启仍保留。
- [ ] 图鉴默认展示所有可展示道具。
- [ ] 图鉴每行 4 张卡片，横向不滚动，纵向滚动。
- [ ] 获得过的道具点亮并保存到 collected_item_ids。
- [ ] 卖出、消耗、升级使用或丢弃道具后，图鉴点亮状态不消失。
- [ ] 保存重载后图鉴点亮状态保留。

## 5. 验收标准

- Debug 可复现满仓和接口消耗。
- Debug 可复现商人出售和货币保存。
- 保存/重载可验证。
- Debug 可复现图鉴点亮、清空、保存重载和出售/消耗后不熄灭。
