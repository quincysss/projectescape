# 09-07 仓库丢弃与确认

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现仓库丢弃，确保玩家明确确认，不静默删除长期资源。

## 2. 丢弃规则

- 从仓库中删除物品。
- 不生成世界掉落。
- 高品质或关键材料需要确认。
- 丢弃写日志。

## 3. 为什么确认

仓库物品是跨局收益，误删代价高。尤其 A/S 品质、前哨材料、研究材料、制作材料应二次确认。

## 4. AI/MCP 开工提示词

```text
请实现仓库丢弃与确认。
要求：
1. 丢弃从 WarehouseData 删除物品。
2. V0.1 不生成世界掉落。
3. A/S 品质、关键材料、研究/制作材料需要确认。
4. 丢弃后保存仓库。
5. Debug 日志记录 item_id、count、quality。
```

## 5. 验收标准

- 普通物品可丢弃。
- 高价值物品有确认。
- 丢弃后仓库容量释放。

## 6. 暂不实现

- 丢弃撤销。
- 回收资源。
