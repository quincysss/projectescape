# 09-14 V0.1 最小交付清单

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 最小闭环

```text
08 结算物品
-> 09 尝试入库
-> 仓库占格/堆叠
-> 满溢返回
-> 玩家整理/丢弃
-> 仓库保存
-> 研究/制作/出发查询
```

## 2. 必须交付

- WarehouseData。
- WarehouseItem。
- WarehouseManager。
- 结算入库接口。
- 手动入库。
- 一键入库优先级。
- 满溢处理。
- 仓库 UI。
- 丢弃确认。
- 堆叠合并。
- 查询/消耗接口。
- 保存/重载。
- Debug 面板。

## 3. AI/MCP 开工提示词

```text
请根据 09_局外仓库系统拆分文档，在 Godot 中完成 V0.1 最小闭环。
优先实现 WarehouseData、WarehouseManager、结算入库、满溢处理、仓库 UI、丢弃确认、堆叠合并、研究/制作/出发查询接口、保存和 Debug。
约束：不要静默丢物；仓库不计算负重；08 入库必须支持部分成功；外部系统不得直接改仓库数组。
```

## 4. 完成定义

- 撤离成功物品能入库。
- 仓库满时剩余物保留。
- 仓库能保存和重载。
- 研究/制作能查询和消耗材料。
- 出发准备能从仓库取物到背包。
