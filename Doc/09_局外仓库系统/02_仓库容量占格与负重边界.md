# 09-02 仓库容量、占格与负重边界

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现局外仓库容量、占格和“不计算负重”的基础规则。

## 2. 初始容量

V0.1 建议仓库比背包大，避免一开始就频繁卡住：

```gdscript
@export var warehouse_width: int = 12
@export var warehouse_height: int = 10
```

## 3. 占格规则

仓库可复用 07 的网格和 ItemDefinition 尺寸：

- 同样检查越界。
- 同样检查重叠。
- 支持堆叠合并。
- V0.1 不做旋转也可接受。

## 4. 负重边界

仓库不计算负重：

- 物品放入仓库后不影响玩家移动。
- 出发准备取出到背包时才重新计算负重。

## 5. AI/MCP 开工提示词

```text
请实现局外仓库容量和占格规则。
要求：
1. 仓库使用与 07 背包兼容的网格逻辑。
2. V0.1 初始容量不要过小，建议 12x10。
3. 仓库物品不计入玩家负重。
4. 从仓库取到背包时才触发 07 WeightComponent 更新。
5. 空间不足时返回明确 reason。
```

## 6. 验收标准

- 仓库可放置多尺寸物品。
- 仓库满时拒绝入库但不吞物。
- 仓库内物品不影响负重。
