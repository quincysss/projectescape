# 09-01 仓库定位与局内存储区别

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

明确局外仓库是长期收益容器，不参与局内负重和风险，不等同于背包、容器残留或前哨临时存储。

## 2. 区别表

| 系统 | 位置 | 生命周期 | 是否长期保存 | 是否计算负重 |
| --- | --- | --- | --- | --- |
| 背包 | 局内 | 当前单局 | 撤离后才保存 | 是 |
| 前哨存储 | 局内 | 当前单局 | 否 | 否/局内临时 |
| 容器残留 | 局内 | 当前地图实例 | 否 | 否 |
| 局外仓库 | 局外 | 跨局 | 是 | 否 |

## 3. AI/MCP 开工提示词

```text
请建立局外仓库系统边界。
要求：
1. 局外仓库只保存撤离成功并确认入库的物品。
2. 仓库不参与局内负重。
3. 前哨临时存储和容器残留不能直接变成仓库，必须经过 08 结算。
4. 研究所、制作所、出发准备只能通过 WarehouseQueryService/WarehouseManager 接口访问。
```

## 4. 验收标准

- 死亡失败不会把局内物品写入仓库。
- 仓库物品跨局保留。
- 仓库和背包数据分离。

## 5. 暂不实现

- 多个仓库地点。
- 仓库被袭击。
