# 09-11 多人/账号预留

> 来源文档：`09_局外仓库系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

V0.1 不做账号服务或多人共享仓库，但预留 owner、profile 和 revision 字段。

## 2. 预留字段

```gdscript
var owner_profile_id: String = "local_profile"
var account_id: String = ""
var warehouse_revision: int = 0
var last_modified_by: String = "local_player"
```

## 3. 未来权威

未来需要权威处理：

- 仓库保存。
- 多设备同步。
- 多人共享仓库权限。
- 消耗材料冲突。
- 入库重复提交。

## 4. AI/MCP 开工提示词

```text
请为局外仓库预留账号/多人字段，但不实现网络同步。
要求：
1. WarehouseData 包含 owner_profile_id、account_id、warehouse_revision、last_modified_by。
2. V0.1 使用 local_profile/local_player。
3. 每次仓库变更 revision +1。
4. 注释说明未来账号服务应拥有保存和冲突解决权威。
```

## 5. 验收标准

- 单机仓库正常保存。
- 数据结构可扩展账号字段。

## 6. 暂不实现

- 账号登录。
- 共享仓库权限。
