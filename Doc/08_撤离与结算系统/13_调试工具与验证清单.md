# 08-13 调试工具与验证清单

> 来源文档：`08_撤离与结算系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. Debug 字段

| 字段 | 用途 |
| --- | --- |
| extraction_active | 撤离是否激活 |
| extraction_progress | 当前撤离进度 |
| end_type | 结束类型 |
| result_built | 是否已生成结算 |
| pending_items | 待处理结算物数量 |
| overflow_items | 溢出物数量 |
| run_seconds | 本局时间 |
| return_loading_mode | 是否正在返回哨所 loading |
| return_loading_progress | 返回哨所 loading 进度 |
| return_ready_to_continue | 是否等待任意按钮 |

## 2. Debug 按钮

- Activate Extraction
- Force Extraction Success
- Force Stability Failed
- Force Timeout Failed
- Build Test Settlement
- Fill Warehouse Then Store
- Clear Settlement
- Force Return To Base Loading
- Force Return Loading Failed
- Complete Return Loading To 100%

## 3. 验证清单

- [ ] 撤离条件未满足不能撤离。
- [ ] 撤离读条可中断。
- [ ] 撤离成功只结算一次。
- [ ] 稳定值归零进入失败。
- [ ] 时间归零进入失败。
- [ ] 成功物品来源正确。
- [ ] 仓库满时结算物不消失。
- [ ] 未处理物品离开前有确认。
- [ ] 成功/失败结算都有返回哨所按钮。
- [ ] 返回哨所 loading 到 100% 后等待任意按钮。
- [ ] 返回哨所 loading 不增加 surface_day。

## 4. AI/MCP 开工提示词

```text
请实现 08 撤离与结算 Debug 面板。
要求：
1. 显示撤离激活、撤离进度、结束类型、是否已生成结算、待处理物、溢出物、本局时间和返回 loading 状态。
2. 提供强制撤离成功、强制稳定失败、强制超时失败、构建测试结算、填满仓库后入库、强制返回 loading 等按钮。
3. 跑完撤离、死亡、结算归属、入库、返回 loading 五类验证清单。
```

## 5. 验收标准

- Debug 可快速复现成功、失败、满仓入库和返回哨所 loading。
- 验证清单覆盖 V0.1 风险。
