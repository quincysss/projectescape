# 08-00 拆分规范与落地文档写法

> 来源文档：`08_撤离与结算系统_修订版_废土生存法则.md`
> 本目录只做落地拆分，不删除、不覆盖原文档。

## 1. 拆分目标

08 模块负责单局如何结束，以及成功/失败后物品和统计如何结算。

落地闭环：

```text
撤离条件满足 -> 撤离点/按钮激活 -> 玩家按住撤离 -> 成功结算
失败条件触发 -> 失败结算
结算物品汇总 -> 玩家选择入库/丢弃 -> 局内状态重置
```

## 3. 施工硬约束

- 成功和失败的损失必须让玩家看懂。
- 入库不能静默替玩家做最终决定。
- 空间不足不能吞物品。
- 失败结算不能把局内收益当作成功收益处理。
- 撤离读条必须可中断。
