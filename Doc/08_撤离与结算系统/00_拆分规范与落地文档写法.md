## 1. 拆分目标

08 模块负责单局如何结束，以及成功/失败后物品和统计如何结算。

落地闭环：

```text
撤离条件满足 -> 撤离点/按钮激活 -> 玩家按住撤离 -> 成功结算
失败条件触发 -> 失败结算
结算物品汇总 -> 结算画面展示 -> 返回哨所 loading -> 按任意键回到局外 -> 局内状态重置
```

## 3. 施工硬约束

- 成功和失败的损失必须让玩家看懂。
- 入库不能静默替玩家做最终决定。
- 空间不足不能吞物品。
- 失败结算不能把局内收益当作成功收益处理。
- 撤离读条必须可中断。
- 成功/失败结算都必须先进入结算画面，再由“返回哨所”按钮进入返回 loading。
- 返回 loading 到 100% 后必须等待任意按钮，不能自动回局外。
