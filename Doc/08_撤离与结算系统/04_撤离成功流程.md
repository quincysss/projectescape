# 08-04 撤离成功流程

> 来源文档：`08_撤离与结算系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

撤离成功后冻结局内输入，收集结算来源，先进入全黑成功结算画面，再等待玩家点击“返回哨所”进入返回 loading。

## 2. 成功流程

```text
撤离进度完成
-> 锁定玩家控制
-> 标记本局结束
-> 屏幕淡入全黑
-> 收集背包/前哨/任务结算来源
-> 构建 RunResult
-> 打开成功 SettlementResultScreen
-> 显示“成功返回404哨所”
-> 展示“本次获得的物资”
-> 玩家点击“返回哨所”
-> 进入 ReturnToBaseLoadingScreen
```

## 3. 需要收集

- 玩家背包物品。
- 前哨临时存储物品。
- 任务/目标奖励。
- 本局统计。

## 4. 信号

```gdscript
signal extraction_succeeded()
signal run_result_built(result: Dictionary)
signal settlement_ui_requested(result: Dictionary)
signal return_to_base_loading_requested(result: Dictionary)
```

## 5. AI/MCP 开工提示词

```text
请实现撤离成功流程。
要求：
1. 撤离完成后立即锁定玩家控制和重复结算入口。
2. 收集 07 背包导出的物品。
3. 收集 06 前哨临时存储 payload。
4. 构建 RunResult，包含物品条目和统计。
5. 打开成功 SettlementResultScreen，标题为“成功返回404哨所”。
6. 展示“本次获得的物资”和返回哨所按钮。
7. 点击返回哨所后进入返回 loading，100% 后等待任意按钮。
8. 成功流程只执行一次。
```

## 6. 验收标准

- 撤离后不能继续操作局内角色。
- 结算物品来源完整。
- 成功结算 UI 能打开，并显示“成功返回404哨所”。
- 点击返回哨所后能进入返回 loading。
