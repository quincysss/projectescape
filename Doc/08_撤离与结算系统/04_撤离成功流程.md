# 08-04 撤离成功流程

> 来源文档：`08_撤离与结算系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `15_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

撤离成功后冻结局内输入，收集结算来源，打开结算界面，并等待玩家处理入库。

## 2. 成功流程

```text
撤离进度完成
-> 锁定玩家控制
-> 标记本局结束
-> 收集背包/前哨/任务结算来源
-> 构建 RunResult
-> 打开 SettlementUI
```

## 3. 需要收集

- 玩家背包物品。
- 前哨临时存储物品。
- 任务/目标奖励。
- 本局统计。

## 4. 信号

```gdscript
signal extraction_succeeded()
signal run_result_built(result: Dictionary)
signal settlement_ui_requested(result: Dictionary)
```

## 5. AI/MCP 开工提示词

```text
请实现撤离成功流程。
要求：
1. 撤离完成后立即锁定玩家控制和重复结算入口。
2. 收集 07 背包导出的物品。
3. 收集 06 前哨临时存储 payload。
4. 构建 RunResult，包含物品条目和统计。
5. 打开 SettlementUI。
6. 成功流程只执行一次。
```

## 6. 验收标准

- 撤离后不能继续操作局内角色。
- 结算物品来源完整。
- 结算 UI 能打开。

- 撤离过场。
