# 25-02 开档剧情与首次出发剧情对白

> 目标：把新档进入游戏后的叙事流程拆成“开场视频、第一段世界观对白、首次出发任务对白”三段，让玩家先理解世界，再在真正点击出发时理解前哨站和撤离规则。

## 1. 最新历程

```text
输入用户名
-> 创建并保存 PlayerProfile
-> 播放开场视频剧情
-> 玩家可跳过视频
-> 视频结束或跳过后，进入第一段剧情对白
-> 第一段对白交代 2050 核战、地下生活、暗潮、异常体质、404 哨所
-> 进入仓库/商人/研究所/出发准备等局外界面
-> 玩家点击出发探索
-> 触发第二段剧情对白
-> 第二段对白交代通道暂时损坏、修复两处前哨站、重新激活撤离
-> 进入出发加载界面
-> 加载完成后停留在 loading 界面，提示“按下任意按钮继续”
-> 玩家按任意按钮
-> 正式进入可操作局内
```

## 2. 档案 flag

新档案推荐使用三个独立标记：

```gdscript
{
    "intro_cinematic_seen": false,
    "world_intro_dialogue_seen": false,
    "first_departure_outpost_dialogue_seen": false
}
```

兼容规则：

- 旧字段 `first_intro_dialogue_seen` 不再作为唯一判断。
- 如果旧档只有 `first_intro_dialogue_seen=true`，迁移时可视为 `world_intro_dialogue_seen=true`。
- `first_departure_outpost_dialogue_seen` 必须单独判断，避免玩家没看过前哨站说明。

## 3. 开场视频剧情

触发条件：

```text
PlayerProfile.intro_cinematic_seen == false
且刚完成用户名创建
```

播放完成或跳过后：

```text
intro_cinematic_seen = true
保存档案
进入第一段世界观对白
```

视频定位：

- 用镜头和字幕交代大时代，不承担具体玩法教学。
- 可以出现核爆、地下避难所、地表废墟、暗潮涌动、404 哨所门牌。
- 视频可以跳过，但跳过视频后仍必须进入第一段世界观对白。
- 跳过按钮文案：`跳过影像`
- 运行时优先播放 MP4：`res://assets/cinematics/source/opening_intro_cinematic_720p.mp4`。
- MP4 播放依赖 `res://addons/ffmpeg/ffmpeg.gdextension`；旧 `.ogv` 可作为兼容回退。

## 4. 第一段剧情：世界观背景

触发条件：

```text
PlayerProfile.world_intro_dialogue_seen == false
且开场视频已结束或被跳过
```

播放完成或跳过后：

```text
world_intro_dialogue_seen = true
保存档案
进入局外仓库/商人/研究所/出发准备界面
```

建议说话人：

| ID | 名称 | 口吻 |
| --- | --- | --- |
| `operator_404` | 404 哨所调度员 | 克制、沉重、任务说明 |
| `player` | 玩家 | 少量回应，不抢信息 |

对白正文：

```text
[operator_404]
欢迎来到 404 哨所。记录显示，你的地表适应反应比大多数人稳定。

[operator_404]
先说清楚现在的世界。

[operator_404]
2050 年，世界爆发了史无前例的核战争。地表城市在火光里断裂，天空连续燃烧了七十多个昼夜。

[operator_404]
幸存下来的人类退入地下。矿井、避难所、旧地铁、军方仓库……能封住门的地方，都成了家。

[operator_404]
但我们从来没有真正放弃地面。

[operator_404]
只是，战争结束后，地表出现了一种新的东西。我们叫它——暗潮。

[operator_404]
它不像烟，也不像水。它无处不在，贴着建筑缝隙流动，钻进呼吸面罩，吞掉暴露在地表的人。

[operator_404]
普通人撑不了多久。几分钟，甚至几十秒，身体和意识就会被暗潮拖走。

[operator_404]
但少数被辐射改变过的人，出现了异常体质。他们能短暂抵御暗潮的侵蚀。

[operator_404]
你就是其中之一。

[player]
所以这就是我被带到这里的理由？

[operator_404]
不是带到这里。是 404 哨所已经没有多少选择了。

[operator_404]
我们的水滤芯快报废了，药品库存不足，主通风井也撑不了太久。

[operator_404]
404 哨所里的每个人，能不能继续活下去，能不能有一天回到地面，全靠你们这些还能走上去的人。

[operator_404]
先去仓库整理一下。等你准备好，再到出发口找我。

[player]
明白。
```

这段对白必须让玩家理解：

- 世界背景是 2050 核战后。
- 人类躲入地下生活。
- 地表存在名为 `暗潮` 的异常物质。
- 普通人无法长时间暴露在地表。
- 玩家因为异常体质可以短暂抵御暗潮。
- 玩家属于 `404 哨所`，地表探索关系到哨所生存。
- 具体前哨和撤离规则暂时不在本段灌输。

## 5. 第二段剧情：首次出发任务说明

触发条件：

```text
PlayerProfile.first_departure_outpost_dialogue_seen == false
且玩家第一次点击“出发探索/确认出发”
且出发校验已通过
```

播放完成或跳过后：

```text
first_departure_outpost_dialogue_seen = true
保存档案
进入 RunLoadingScreen
```

对白正文：

```text
[operator_404]
出发前最后确认一次任务目标。

[operator_404]
地表的旧通道已经不能稳定往返。你上去以后，升降通道会被暗潮反冲干扰，回来的路会暂时损坏。

[player]
也就是说，我不能直接回来？

[operator_404]
不能。至少不能立刻回来。

[operator_404]
你需要在地表找到前哨站材料。我们已经把它们标成菱形信号，靠近后会更容易识别。

[operator_404]
收集足够材料后，修复两处前哨站。前哨站会重新建立地面信号链。

[operator_404]
两处前哨都修好后，通道才能被重新激活，撤离信号才会亮起。

[player]
如果没来得及修好呢？

[operator_404]
暗潮会不断压缩你的视野，也会让你的稳定值持续下降。

[operator_404]
稳定值太低时，你会越来越难判断周围情况。不要贪，物资再多也得活着带回来。

[operator_404]
记住顺序：先找菱形材料，修复前哨站，再撤离。

[operator_404]
回到基地后，你的稳定值会恢复。那里也是你整理物资、出售战利品和准备下一次探索的地方。

[player]
收到。先找材料，修前哨，再回来。

[operator_404]
祝你好运。404 哨所等你回来。
```

这段对白必须让玩家理解：

- 点击出发后才讲具体任务规则。
- 出发后通道会暂时损坏，撤离不是开局可用。
- 玩家需要先收集菱形前哨材料。
- 修复两处前哨站后，撤离才会重新激活。
- 暗潮会压缩视野。
- 稳定值会随在外时间下降。
- 回到基地后稳定值能恢复。

## 6. UI/交互规则

- 开场视频可跳过，跳过后进入第一段世界观对白。
- 第一段和第二段对白支持点击/按键推进。
- V0.1 可以保留对白跳过，但跳过只跳过当前对白，不跳过后续流程。
- 每句最长不超过两行。
- 重要词高亮：`暗潮`、`404 哨所`、`菱形材料`、`两处前哨站`、`撤离信号`、`稳定值`。
- 第一段对白结束后进入局外界面。
- 第二段对白结束后进入 loading。
- 第二段对白只在第一次出发探索时播放一次。

## 7. AI/MCP 开工提示词

```text
请实现开档剧情与首次出发剧情流程。

要求：
1. 玩家输入用户名并创建 PlayerProfile 后，播放开场视频剧情；视频可跳过。
2. 视频结束或跳过后，播放 world_intro_dialogue，交代 2050 核战、地下生活、暗潮、异常体质和 404 哨所。
3. world_intro_dialogue 完成后进入局外仓库/商人/研究所/出发准备界面。
4. 玩家第一次点击出发探索且出发校验通过后，播放 first_departure_outpost_dialogue。
5. first_departure_outpost_dialogue 讲清楚：通道暂时损坏、收集菱形材料、修复两处前哨站、撤离重新激活、视野收缩、稳定值下降、回基地恢复稳定值。
6. 第二段对白完成或跳过后进入 RunLoadingScreen。
7. 使用 DialogueSequence 数据驱动对白，不要把对白硬编码在 UI 脚本里。
8. 保存 intro_cinematic_seen、world_intro_dialogue_seen、first_departure_outpost_dialogue_seen 三个 flag。
```

## 8. 验收标准

- 新档输入用户名后播放开场视频。
- 跳过视频后仍进入第一段世界观对白。
- 第一段对白结束后进入局外仓库等界面，而不是直接进局。
- 玩家第一次点击出发探索后播放第二段任务对白。
- 第二段对白结束后进入 loading。
- 两段对白不会重复播放。
- 玩家能从第二段对白理解“菱形材料 -> 修复前哨站 -> 解锁撤离”的顺序。
