# 25-03 首次返回剧情与第一章目标

> 目标：第一次成功撤离回家后，触发玩家独白，并把第一章目标设为“为救出妹妹，解锁制造所”。

## 1. 触发条件

```text
PlayerProfile.first_return_dialogue_seen == false
且本局结束类型 == EXTRACTION_SUCCESS
且结算画面点击返回哨所
且返回哨所 loading 已到 100%
且玩家已按任意按钮回到局外界面
```

播放完成后：

```text
first_return_dialogue_seen = true
chapter_1_goal_active = true
保存档案
```

## 2. 演出形式

V0.1 推荐：

- 仓库或地下住处背景。
- 玩家独白，无需新 NPC。
- 画面可短暂聚焦商人页签或矿币 UI。
- 对白结束后弹出第一章目标卡。
- 独白中出现的道具名必须来自 `setting/items.tab`。
- 本段当前引用的道具示例：`废金属`、`旧线圈`、`药粉`、`工具零件`、`安定糖`、`临时绷带`、`褪色照片`、`金色数据芯片`、`旧电池`。

## 3. 对白正文

```text
[player]
回来了……

[player]
门关上的声音，听起来比地表的风声踏实多了。

[player]
仓库终端刚把清单扫完。

[player]
这些东西，商人那边一定有人会要。

[player]
先去商人那看看吧。能卖的就换成矿币，剩下的交给仓库和研究所。

[player]
毕竟，现在对我来说，矿币才是最重要的。

[player]
他们说的旧时代制造机，说不定真能做出救她要用的东西……

[player]
如果是真的，妹妹就还有救。

[player]
得尽快攒够矿币，解锁制造所。

[player]
这是救出她的第一步。
```

## 4. 第一章目标卡

对白结束后弹出：

```text
第一章目标
解锁制造所

出售可售物资，积攒 5000 矿币。
制造所解锁后，才有机会推进救出妹妹的计划。
```

按钮：

```text
知道了
前往商人
```

如果玩家点击 `前往商人`：

```text
切换到局外商人页签
```

## 5. 目标追踪规则

局外界面顶部或侧边显示章节目标：

```text
第一章：解锁制造所
矿币 0 / 5000
```

当玩家出售物品获得矿币时刷新进度。

## 6. AI/MCP 开工提示词

```text
请实现首次成功撤离返回后的剧情独白和第一章目标。

要求：
1. 触发条件为 first_return_dialogue_seen == false、本局成功撤离、结算完成、返回哨所 loading 完成且玩家按任意按钮回到局外界面。
2. 使用 DialogueSequence 数据驱动 25-03 文档中的 player 独白。
3. 播放完成后 first_return_dialogue_seen=true，chapter_1_goal_active=true，并保存档案。
4. 弹出第一章目标卡：为救出妹妹，解锁制造所；出售可售物资并积攒 5000 矿币。
5. 目标卡提供“知道了”和“前往商人”按钮。
6. “前往商人”切换到局外商人页签。
7. 局外界面显示章节目标进度：矿币 current / 5000。
```

## 7. 验收标准

- 第一次成功撤离返回后播放独白。
- 失败返回不播放该独白。
- 播放后不重复。
- 第一章目标显示并随矿币变化刷新。
- 点击前往商人能进入商人页签。
- 独白中的道具名均能在 `setting/items.tab` 中找到。
- 独白能明确表达：解锁制造所是救出妹妹的第一步。
