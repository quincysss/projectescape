# 25-03 首次返回剧情与第一章目标

> 目标：第一次成功撤离回家后，触发玩家独白，并把第一章目标设为“为救出妹妹，解锁制造所”。

## 1. 触发条件

```text
PlayerProfile.first_return_dialogue_seen == false
且本局结束类型 == EXTRACTION_SUCCESS
且结算画面点击返回哨所
且返回哨所 loading 已到 100%
且玩家已按任意按钮回到局外界面
```

播放完成后：

```text
first_return_dialogue_seen = true
merchant_unlocked = true
research_station_unlocked = true
chapter_1_goal_active = true
保存档案
```

## 2. 演出形式

V0.1 推荐：

- 仓库或地下住处背景。
- 玩家独白，无需新 NPC，只显示主角原画。
- 画面可短暂聚焦商人页签或矿币 UI。
- 对白结束后解锁商人和研究所，再弹出第一章目标卡。
- 独白中出现的道具名必须来自 `setting/items.tab`。
- 本段当前引用的道具示例：`废金属`、`旧线圈`、`药粉`、`工具零件`、`安定糖`、`临时绷带`、`褪色照片`、`金色数据芯片`、`旧电池`。

原画显示：

- 本段全部为 `player` 独白。
- DialoguePanel 只显示主角原画，默认位于右侧或居中偏右。
- 不显示 404 哨所管理员原画，避免误以为有 NPC 在场。
- 主角原画配置来自 `setting/dialogue_speakers.tab`，详细规则见 `08_剧情对白角色原画显示规则.md`。

## 4. 第一章目标卡

对白结束后弹出：

```text
第一章目标
解锁制造所

出售可售物资，积攒 5000 矿币。
制造所解锁后，才有机会推进救出妹妹的计划。
```

按钮：

```text
知道了
前往商人
```

解锁规则：

```text
first_return_chapter_1_dialogue 播放完成
-> BaseFeatureUnlockService.unlock("merchant")
-> BaseFeatureUnlockService.unlock("research_station")
-> 保存 PlayerProfile
-> 刷新局外顶部页签
-> 弹出第一章目标卡
```

如果玩家点击 `前往商人`：

```text
确认 merchant_unlocked == true
切换到局外商人页签
```

第一天首次进入局外时，商人和研究所不开放。顶部页签可以保留固定位置，但必须显示锁定或置灰状态，并禁止点击进入功能界面。锁定文案建议：

```text
商人：等待第一次地表回收后开放
研究所：等待第一次地表回收后开放
```

## 5. 目标追踪规则

局外界面顶部或侧边显示章节目标：

```text
第一章：救出妹妹
目标：解锁制造所
矿币 0 / 5000
```

当玩家出售物品获得矿币时刷新进度。

## 6. AI/MCP 开工提示词

```text
请实现首次成功撤离返回后的剧情独白和第一章目标。

要求：
1. 触发条件为 first_return_dialogue_seen == false、本局成功撤离、结算完成、返回哨所 loading 完成且玩家按任意按钮回到局外界面。
2. 使用 `setting/dialogues.tab` 数据驱动 25-03 文档中的 player 独白，修改对白只改表。
3. 独白播放时只显示主角原画，不显示 404 哨所管理员。
4. 播放完成后 first_return_dialogue_seen=true、merchant_unlocked=true、research_station_unlocked=true、chapter_1_goal_active=true，并保存档案。
5. 新档第一天首次进入局外时，merchant_unlocked=false、research_station_unlocked=false，商人和研究所页签置灰锁定且不可点击。
6. 首次返回剧情结束后刷新局外顶部页签，商人和研究所变为可用。
7. 弹出第一章目标卡：为救出妹妹，解锁制造所；出售可售物资并积攒 5000 矿币。
8. 目标卡提供“知道了”和“前往商人”按钮。
9. “前往商人”必须在 merchant_unlocked=true 后切换到局外商人页签。
10. 局外界面显示章节目标进度：矿币 current / 5000。
```

## 7. 验收标准

- 第一次成功撤离返回后播放独白。
- 失败返回不播放该独白。
- 播放后不重复。
- 第一章目标显示并随矿币变化刷新。
- 首次返回剧情结束前，商人和研究所仍未解锁。
- 首次返回剧情结束后，商人和研究所解锁并刷新页签状态。
- 点击前往商人能进入商人页签。
- 独白中的道具名均能在 `setting/items.tab` 中找到。
- 独白能明确表达：解锁制造所是救出妹妹的第一步。
- 独白期间只显示主角原画，结束后原画隐藏。
