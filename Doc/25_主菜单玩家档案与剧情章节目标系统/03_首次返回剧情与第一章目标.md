# 25-03 首次返回剧情与第一章目标

> 目标：第一次成功撤离回家后，触发玩家独白，并把第一章目标设为解锁制造所。

## 1. 触发条件

```text
PlayerProfile.first_return_dialogue_seen == false
且本局结束类型 == EXTRACTION_SUCCESS
且结算画面点击返回哨所
且返回哨所 loading 已到 100%
且玩家已按任意按钮回到局外界面
```

播放完成后：

```text
first_return_dialogue_seen = true
chapter_1_goal_active = true
保存档案
```

## 2. 演出形式

V0.1 推荐：

- 仓库或地下住处背景。
- 玩家独白，无需新 NPC。
- 画面可短暂聚焦商人页签或矿币 UI。
- 对白结束后弹出第一章目标卡。

## 3. 对白正文

```text
[player]
回来了……

[player]
通道还能撑住，至少这一次撑住了。

[player]
这些东西在地面上只是废铁、旧零件、被暗潮泡过的垃圾。

[player]
可在下面，它们能换水、换药，换下一次上去的机会。

[player]
也许……如果把带回来的物资卖给黑市的商人，能换到更多矿币。

[player]
有了足够的矿币，我就能买下那台旧制造机。

[player]
如果制造所能重新运转，我们就不用只等别人施舍零件。

[player]
那样的话……

[player]
妹妹也许还能有机会。
```

## 4. 第一章目标卡

对白结束后弹出：

```text
第一章目标
解锁制造所

出售从地面带回的物资，积攒 5000 矿币。
制造所解锁后，404 哨所将获得基础制造能力。
```

按钮：

```text
知道了
前往商人
```

如果玩家点击 `前往商人`：

```text
切换到局外商人页签
```

## 5. 目标追踪规则

局外界面顶部或侧边显示章节目标：

```text
第一章：解锁制造所
矿币 0 / 5000
```

当玩家出售物品获得矿币时刷新进度。

## 6. AI/MCP 开工提示词

```text
请实现首次成功撤离返回后的剧情独白和第一章目标。

要求：
1. 触发条件为 first_return_dialogue_seen == false、本局成功撤离、结算完成、返回哨所 loading 完成且玩家按任意按钮回到局外界面。
2. 使用 DialogueSequence 数据驱动 25-03 文档中的 player 独白。
3. 播放完成后 first_return_dialogue_seen=true，chapter_1_goal_active=true，并保存档案。
4. 弹出第一章目标卡：解锁制造所，积攒 5000 矿币。
5. 目标卡提供“知道了”和“前往商人”按钮。
6. “前往商人”切换到局外商人页签。
7. 局外界面显示章节目标进度：矿币 current / 5000。
```

## 7. 验收标准

- 第一次成功撤离返回后播放独白。
- 失败返回不播放该独白。
- 播放后不重复。
- 第一章目标显示并随矿币变化刷新。
- 点击前往商人能进入商人页签。
