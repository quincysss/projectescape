# 25-03 序章返回剧情与第一章目标

> 目标：序章探索结束返回哨所后，无论玩家撤离成功或失败，都触发返回剧情，并通过剧情包装发放一次性开店周转物资，确保玩家有材料完成首次制造、上架和开店结算。

## 1. 触发条件

```text
PlayerProfile.first_return_dialogue_seen == false
且本局为序章首次探索结算
且结算画面点击返回哨所
且返回哨所 loading 已到 100%
且玩家已按任意按钮回到局外界面
```

本段不要求 `EXTRACTION_SUCCESS`。允许触发的结算类型：

| 结算类型 | 是否触发 | 说明 |
| --- | --- | --- |
| `EXTRACTION_SUCCESS` | 是 | 玩家成功撤离，剧情承认玩家带回了物资 |
| `RUN_FAILED` | 是 | 玩家未能成功撤离，剧情改用哨所凑出的周转物资兜底 |
| `PLAYER_ABORT_DEBUG` | 否 | Debug 或开发中断不触发正式剧情 |

播放完成后：

```text
first_return_dialogue_seen = true
shop_loop_unlocked = true
research_station_unlocked = true
starter_shop_supply_granted = true
chapter_1_goal_active = true
保存档案
发放一次性开店周转物资
刷新局外昼夜阶段入口
弹出第一章目标卡
```

如果 `starter_shop_supply_granted == true`，不得重复发放周转物资。

## 2. 演出形式

V0.1 推荐：

- 背景：杂货店门口、半亮的货台、仓库侧门或地下通道尽头。
- 角色：主角与 404 哨所管理员。
- 画面可短暂扫过落灰的店招、空货台、旧灯管、送来的散料箱，不要像功能教学镜头。
- 对白结束后开启正式白天经营循环和研究所，再弹出第一章目标卡。
- 对白中出现的道具名必须来自 `setting/items.tab`。
- 本段当前引用的道具示例：`废金属`、`脏布条`、`旧电池`、`旧线圈`、`工具零件`、`安定糖`、`临时绷带`。

剧情写法要求：

- 对白只承担情绪、关系和动机，不直接讲“需求榜排名”“成交间隔”“步骤 1/2/3”。
- 教学信息放在第一章目标卡、UI 指引和规则段，不放进角色嘴里。
- 404 哨所管理员的口吻仍然克制、熟悉、少废话；他可以解释现实压力，但不能像系统播报。
- 玩家对杂货店重新开门应有迟疑、负担感和一点自嘲，而不是立刻接受教程任务。
- 周转物资要包装成人情债、店账、居民寄放或哨所仓底旧料，不表现为系统补偿。

原画显示：

- `operator_404` 说话时显示 404 哨所管理员原画。
- `player` 说话时显示主角原画。
- 两者均位于屏幕最左侧并左对齐，按当前 `speaker_id` 切换。
- 主角原画配置来自 `setting/dialogue_speakers.tab`，详细规则见 `08_剧情对白角色原画显示规则.md`。

## 3. 对白正文：成功撤离版本

```text
[operator_404]
你回来了。

[player]

[operator_404]
这次收获怎么样。

[player]
只有这些材料了……听起来不像什么好消息。

[operator_404]
店还得重新打理，大家也还在等着。

[operator_404]
制造机能启动吗？

[operator_404]
我已经让人预热好了。

[player]
好，这就开工。这些材料，应该也够做出一点东西。
```

## 4. 对白正文：撤离失败或空包版本

```text
[operator_404]
你回来了。

[operator_404]
仓底还有一点旧电池和工具零件。我签了出库。

[operator_404]
加上你带回来的东西，应该够撑过今天开店了。

[player]
制造机已经预热好了吗？

[operator_404]
嗯，好了。

[player]
那开工吧，大伙还在等着我。


```

## 5. 一次性开店周转物资

周转物资定位：

- 不是系统补偿，而是剧情包装下的“404 哨所与居民凑出的第一天开店本钱”。
- 只发一次，用于防止序章失败导致玩家没材料、没钱、无法开店。
- 发放后进入仓库，玩家仍需手动在制造所加工成可售物资。
- 周转物资不直接给可售物资，避免跳过制造教学。

建议发放内容：

| 道具类型 | 建议数量 | 用途 |
| --- | --- | --- |
| C级基础金属类道具 | 2 | 支撑基础五金、修补件类可售物资 |
| C级布料/纤维类道具 | 2 | 支撑绷带、过滤布、包扎类可售物资 |
| C级塑料/胶片类道具 | 1 | 支撑简易防护或包装类可售物资 |
| C级电气/线圈类道具 | 1 | 支撑小型照明、电池夹、零件类可售物资 |

落地时以 `setting/items.tab` 的真实 item_id 为准。若当前表中缺少对应分类，应先按 Doc/30/31 的新物资规划补表，再接入本奖励。

伪代码：

```gdscript
if not profile.starter_shop_supply_granted:
    StarterSupplyService.grant("prologue_shop_starter_pack")
    profile.starter_shop_supply_granted = true
    profile.shop_loop_unlocked = true
    profile.research_station_unlocked = true
    profile.chapter_1_goal_active = true
    ProfileService.save(profile)
```

## 6. 第一章目标卡

对白结束后弹出：

```text
第一章目标
重启杂货店

使用周转物资或带回的材料，在制造所加工第一批可售物资。
查看今日需求榜，把货摆上货台，完成第一次营业结算。
```

按钮：

```text
知道了
前往制造所
查看需求榜
```

如果玩家点击 `前往制造所`：

```text
确认 shop_loop_unlocked == true
进入白天准备阶段制造所界面
```

如果玩家点击 `查看需求榜`：

```text
确认 shop_loop_unlocked == true
进入白天准备阶段需求榜界面
```

## 7. 目标追踪规则

局外界面顶部或侧边显示章节目标：

```text
第一章：重启杂货店
目标：完成首次营业
步骤 1/3：制造第一批可售物资
步骤 2/3：将可售物资摆上货台
步骤 3/3：完成第一次开店结算
```

目标推进：

| 事件 | 目标变化 |
| --- | --- |
| 任意可售物资制造成功 | 步骤 1 完成 |
| 任意可售物资上架到货台 | 步骤 2 完成 |
| 第一次开店结算面板关闭 | 步骤 3 完成，`first_shop_tutorial_completed=true` |

矿币仍然重要，但不作为第一章硬性数值目标。后续矿币主要用于出发交通、装备与消耗品补给、研究升级、扩展货台、扩展配方等消耗。

## 8. AI/MCP 开工提示词

```text
请实现序章返回后的剧情、一次性开店周转物资和第一章目标。

要求：
1. 触发条件为 first_return_dialogue_seen == false、本局为序章首次探索结算、结算完成、返回哨所 loading 完成且玩家按任意按钮回到局外界面。
2. 不要求成功撤离；撤离成功、撤离失败或空包返回都要进入该剧情，但对白分支不同。
3. 使用 `setting/dialogues.tab` 数据驱动 25-03 文档中的剧情对白，修改对白只改表。
4. 播放时按 speaker_id 显示主角和 404 哨所管理员原画，立绘统一左侧。
5. 播放完成后 first_return_dialogue_seen=true、shop_loop_unlocked=true、research_station_unlocked=true、starter_shop_supply_granted=true、chapter_1_goal_active=true，并保存档案。
6. 如果 starter_shop_supply_granted 已经为 true，不得重复发放周转物资。
7. 周转物资发入仓库，不能直接给可售物资；玩家仍需通过制造所加工。
8. 新第一章目标卡为“重启杂货店”，要求制造第一批可售物资、上架货台、完成第一次开店结算。
9. 目标卡提供“知道了”“前往制造所”“查看需求榜”按钮。
10. “前往制造所”和“查看需求榜”必须在 shop_loop_unlocked=true 后切换到白天准备阶段对应界面。
11. 不显示旧独立交易页签，不写额外制造设施目标，不写固定矿币章节目标。
```

## 9. 验收标准

- 第一次序章探索返回后播放返回剧情。
- 成功撤离会播放成功撤离版本对白。
- 撤离失败或空包返回会播放兜底版本对白。
- 播放后不重复。
- 周转物资只发一次。
- 周转物资进入仓库，不能绕过制造所直接成为可售物资。
- 首次返回剧情结束前，正式白天经营循环仍未开放。
- 首次返回剧情结束后，正式白天经营循环和研究所解锁并刷新入口状态。
- 点击前往制造所能进入白天准备阶段制造所界面。
- 对白中的道具名均能在 `setting/items.tab` 中找到，或在物资表重构任务中先补齐。
- 第一章目标显示为重启杂货店和首次营业，不显示额外制造设施或固定矿币目标。
