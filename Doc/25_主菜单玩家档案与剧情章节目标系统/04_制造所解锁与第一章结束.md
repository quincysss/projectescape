# 25-04 制造所默认开放与首次开店教学

> 目标：替换旧的“制造设施分层/固定矿币解锁”章节规则。最新流程中，制造所是默认可用的局外设施，第一章重点是让玩家完成“材料 -> 制造 -> 上架 -> 营业结算”的首次经营闭环。

## 1. 废弃规则

以下规则不再作为正向设计：

- 第一章赚固定数额矿币解锁额外制造设施。
- 以额外制造设施作为救出妹妹的第一章终点。
- 制造所本体需要通过矿币解锁。
- 失败撤离不触发首次返回剧情。
- 玩家无物资时等待玩家再次出发后才能开店。

若旧代码或旧文档中仍有上述逻辑，后续实现应按本文和 `03_首次返回剧情与第一章目标.md` 替换。

## 2. 最新制造所开放规则

```text
新档创建
-> 制造所设施在世界设定上已经存在
-> 序章首次出发前不进入白天经营循环
-> 序章探索结束返回局外
-> 剧情发放一次性开店周转物资
-> shop_loop_unlocked = true
-> 白天准备阶段开放制造所入口
```

制造所入口状态：

| 阶段 | UI 状态 | 说明 |
| --- | --- | --- |
| 开档后、序章出发前 | 可隐藏或弱化展示 | 玩家当前只需要完成序章夜晚出发 |
| 序章返回剧情前 | 不允许进入白天准备阶段 | 避免玩家在没有周转物资前卡流程 |
| 序章返回剧情后 | 可点击 | 玩家可用周转物资制造可售物资 |
| 开店阶段 | 可按局外白天经营文档限制 | 若允许开店中制造，必须保证不破坏营业节奏 |

## 3. 首次开店教学目标

第一章目标拆成三个可验证步骤：

```text
步骤 1：制造第一批可售物资
步骤 2：将可售物资摆上货台
步骤 3：完成第一次开店结算
```

完成后：

```text
first_shop_tutorial_completed = true
chapter_1_completed = true
保存档案
显示第一章阶段完成提示
进入常规昼夜循环
```

阶段完成提示：

```text
第一天营业结束

杂货店重新开门了。
避难所开始知道，你这里不只收废料，也能把废料变成他们真正需要的东西。

接下来，白天经营店铺，晚上前往地表。
用赚到的矿币扩展货台、购买出行补给、研究升级，并准备去更远的地点。
```

按钮：

```text
继续
前往夜间计划
```

## 4. 周转物资与制造配方要求

周转物资必须能稳定合成至少 1 种 C 级可售物资，并尽量支持 2-3 种选择，避免教学过窄。

推荐规则：

- 周转包不给可售物资本体，只给 C 级基础道具。
- 首批可制作物资应以 C 级为主。
- 首批配方材料种类以 1-3 种为主。
- 周转包材料总量要能保证玩家至少做出 3 件可售物资，用于填充初始 3 个货台格。
- 若玩家序章成功带回额外材料，系统不额外削减周转包；多余材料作为玩家表现奖励。

示例教学配方方向：

| 可售物资方向 | 材料组合方向 | 教学目的 |
| --- | --- | --- |
| 简易修补包 | 金属 + 布料 | 让玩家理解多材料合成 |
| 过滤布卷 | 布料 + 塑料/胶片 | 对应避难所基础生活需求 |
| 小型照明件 | 电气/线圈 + 金属 | 引出电气类材料价值 |

具体 item_id、sale_good_id 和产出数量以物资表重构后的配置为准。

## 5. 失败与原子性

周转物资发放必须原子化：

```text
检查 starter_shop_supply_granted
-> 写入仓库物品
-> 设置 starter_shop_supply_granted=true
-> 设置 shop_loop_unlocked=true
-> 保存档案和仓库
```

如果保存失败：

- 不显示目标卡成功推进。
- 不进入正式白天经营循环。
- 提示保存失败或重试。
- 不得出现 flag 已写入但仓库未获得周转物资的状态。

首次开店教学完成也必须原子化：

```text
完成第一次开店结算
-> 结算货币和库存变化
-> first_shop_tutorial_completed=true
-> chapter_1_completed=true
-> 保存
-> 显示阶段完成提示
```

## 6. AI/MCP 开工提示词

```text
请按最新流程实现制造所默认开放和首次开店教学。
要求：
1. 不实现额外制造设施作为第一章目标，不实现固定矿币解锁制造所。
2. 制造所设施默认存在，但玩家序章返回剧情前不进入白天准备阶段。
3. 序章返回剧情结束后，使用 StarterSupplyService 发放一次性周转物资，开启 shop_loop_unlocked。
4. 周转物资进入仓库，玩家必须在制造所把材料加工成可售物资。
5. 第一章目标为制造第一批可售物资、上架货台、完成第一次开店结算。
6. 完成首次开店结算后设置 first_shop_tutorial_completed=true、chapter_1_completed=true。
7. 阶段完成提示引导玩家进入常规昼夜循环和夜间计划。
8. 任何旧的固定矿币解锁、额外制造设施入口、旧独立交易入口都不得在新手主流程中出现。
```

## 7. 验收标准

- 新档流程不要求玩家攒固定矿币解锁制造所。
- 序章返回剧情后制造所入口可用。
- 序章失败且空包返回时，周转物资仍能支撑首次制造。
- 周转物资只发一次。
- 玩家能用周转物资制造至少 1 种 C 级可售物资。
- 玩家完成制造、上架、开店结算后，第一章教学目标完成。
- UI 中不出现额外制造设施解锁按钮或固定矿币目标。
