# 07-02 背包格子与占格规则

> 来源文档：`07_背包_存储与负重系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `16_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现网格背包：物品按尺寸占用格子，系统能判断可放置、放置、移除和查找空位。

## 2. 背包容量

V0.1 可先使用固定背包网格：

```gdscript
@export var width: int = 8
@export var height: int = 6
```

后续可由背包装备或角色属性扩展。

## 3. 坐标规则

坐标从左上角开始：

```text
(0,0) -> (width-1, 0)
...
(0,height-1)
```

ItemStack 记录左上角坐标：

```gdscript
var origin: Vector2i
var size: Vector2i
```

## 4. 占格判断

必须提供：

```gdscript
func can_place(stack: ItemStack, origin: Vector2i) -> bool
func place_stack(stack: ItemStack, origin: Vector2i) -> bool
func remove_stack(stack_id: String) -> bool
func find_first_fit(stack: ItemStack) -> Vector2i
```

规则：

- 不能越界。
- 不能覆盖其他物品。
- 同一堆叠占据矩形区域。
- 旋转 V0.1 可不做。

## 5. AI/MCP 开工提示词

```text
请实现背包网格与占格规则。

要求：
1. InventoryGrid 支持 width/height。
2. ItemStack 记录 origin 和 size。
3. 实现 can_place、place_stack、remove_stack、find_first_fit。
4. 放置时检查越界和占用。
5. V0.1 不做旋转。
6. 所有失败返回明确 reason，不要静默失败。
```

## 6. 验收标准

- 1x1、1x2、2x2 物品能正确占格。
- 越界放置失败。
- 重叠放置失败。
- 移除后格子释放。

## 7. 暂不实现

- 物品旋转。
- 自动整理动画。
