# 07-14 调试工具与验证清单

> 来源文档：`07_背包_存储与负重系统_修订版_废土生存法则.md`

## 1. Debug 面板字段

| 字段 | 用途 |
| --- | --- |
| inventory_slots_used | 背包占用格 |
| total_weight | 当前总负重 |
| weight_stage | 负重阶段 |
| selected_stack | 选中物品堆 |
| home_storage_used | 家中存储占用 |
| temporary_storage_count | 临时存储数量 |
| revision | 背包版本号 |

## 2. Debug 按钮

- Add Heavy Items

- Create Temp Storage

- Simulate Extraction

## 3. 验证清单

### 背包

- [ ] 物品可放入空格。
- [ ] 大尺寸物品占多格。
- [ ] 堆叠优先合并。
- [ ] 空间不足不吞物。

### 负重

- [ ] 背包重量计算正确。
- [ ] 装备重量计入。
- [ ] 超重仍可拾取。
- [ ] 移动惩罚接口可读。

### 存储

- [ ] 家中存储可双向转移。
- [ ] 前哨临时存储可创建和销毁。
- [ ] 空间不足不丢物。

### 结算

- [ ] 死亡清理局内背包。
- [ ] 撤离导出物品。
- [ ] 家中存储保留。

## 4. AI/MCP 开工提示词

```text
请实现 07 背包/存储/负重 Debug 面板和验证辅助。

要求：
1. 显示背包占格、总负重、负重阶段、选中堆叠、家中存储占用、临时存储数量、revision。
2. 提供添加测试物品、填满背包、清空背包、添加重物、创建临时存储、模拟死亡、模拟撤离等按钮。
3. 所有失败操作输出 reason。
4. 跑完背包、负重、存储、结算四类验证清单。
```

## 5. 验收标准

- Debug 能复现满包、超重、转移失败、死亡撤离。
- 验证清单覆盖 05 和 06 的依赖接口。
