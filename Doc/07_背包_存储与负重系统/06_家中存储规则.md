# 07-06 家中存储规则

> 来源文档：`07_背包_存储与负重系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `16_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

实现家中存储作为主要安全仓库，用于撤离成功后的物品整理和长期资源保留。

## 2. 容量规则

V0.1 可以使用固定容量：

```gdscript
@export var home_storage_width: int = 10
@export var home_storage_height: int = 8
```

后续可通过家园升级扩容。

## 3. 存储规则

- 家中存储为长期存储。
- 可从背包转入。
- 可从家中取出到背包。
- 取出时必须检查背包空间和负重提示。
- 玩家进入家安全区时不自动打开背包或家中存储。
- 家中撤离信号 UI 下方显示“按 TAB 打开背包界面”提示。
- 玩家在家中按 TAB 时打开背包 + 家中存储双栏。
- 玩家离开家安全区时，如果家中背包/存储界面打开，则自动关闭。
- 特殊任务物品可设置禁止存入。

## 4. AI/MCP 开工提示词

```text
请实现家中存储。

要求：
1. 家中存储使用与背包兼容的 InventoryGrid 或 StorageContainer。
2. V0.1 使用固定容量。
3. 支持背包到家中、家中到背包的转移。
4. 取出失败时物品不得消失。
5. 家中存储是长期存储，与前哨临时存储区分。
```

## 5. 验收标准

- 撤离成功物品可进入家中存储。
- 背包和家中存储可以双向转移。
- 背包满时取出失败且物品保留。

## 6. 暂不实现

- 家园升级扩容。
- 分类仓库。
