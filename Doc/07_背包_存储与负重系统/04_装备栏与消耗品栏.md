# 07-04 装备栏与消耗品栏

> 来源文档：`07_背包_存储与负重系统_修订版_废土生存法则.md`
> 施工补充：开工前必须读取 `16_模块功能规则细化与AI开工提示词.md`。

## 1. 实现目标

为玩家提供基础装备栏和消耗品快捷栏，并让装备重量纳入负重系统。

## 2. 装备栏

V0.1 建议槽位：

```text
weapon
armor
tool
backpack
accessory
```

装备规则：

- 只有 `item_type == equipment` 的物品可装备。
- 装备槽类型必须匹配。
- 装备后仍计入总负重。
- 卸下装备需要背包有空间。

## 3. 消耗品栏

V0.1 建议 3-4 个快捷槽：

```text
quick_1
quick_2
quick_3
quick_4
```

规则：

- 只允许 `consumable` 或白名单物品放入。
- 使用时从对应堆叠扣数量。
- 数量为 0 时清空快捷槽。

## 4. AI/MCP 开工提示词

```text
请实现基础装备栏与消耗品快捷栏。

要求：
1. 装备槽至少包含 weapon、armor、tool、backpack、accessory。
2. 装备物品必须 item_type 为 equipment 且槽位匹配。
3. 装备重量计入总负重。
4. 卸下装备时必须检查背包空间。
5. 快捷栏支持 3-4 个消耗品槽，使用后扣除数量。
6. 不实现复杂装备属性，只保留接口。
```

## 5. 验收标准

- 不匹配装备不能放入槽位。
- 装备后负重变化。
- 背包满时不能卸下装备导致物品消失。
- 消耗品使用后数量减少。

## 6. 暂不实现

- 装备耐久。
- 装备词缀。
- 快捷栏拖拽动画。
