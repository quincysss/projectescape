## 1. 拆分目标

07 是多个系统的底座：容器拾取、前哨材料、前哨存储、死亡撤离结算都会依赖背包/存储/负重规则。

落地目标：

```text
物品定义 -> 物品堆叠 -> 背包格子 -> 装备/快捷栏 -> 负重计算 -> 家/前哨存储 -> 物品转移 -> 死亡/撤离归属
```

## 3. 施工硬约束

- 拾取失败不能吞物品。
- 背包满不能让容器残留物消失。
- 超重不应直接阻止拾取，先警告和惩罚移动。
- 前哨存储不是永久仓库。
- 丢弃规则 V0.1 不生成世界掉落，避免刷物和复杂拾回。
- 所有外部系统通过接口请求背包，不要复制物品逻辑。
