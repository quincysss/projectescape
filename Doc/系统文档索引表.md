## 4. 后续维护规则

- 新增系统目录时，必须新增该目录的 `00_AI开工入口.md`。
- 后续需求文档整理必须按 `Doc/00_BDD需求文档规范.md` 补充 `Feature / Scenario / Given / When / Then`。
- 新增顶层专题文档时，必须补入“专题文档路由”。
- 拆分旧专题为系统目录时，必须把路由从“专题文档”移动到“系统路由表”。
- 如果系统验收清单或 AI 提示词改名，必须同步本表、目标系统入口、skill 和 doc-map。
