## 2. 稳定值核心组件

对应文档：`01_稳定值核心组件.md`

### 功能规则细化

| 分类 | 规则 |
|---|---|
| 前置条件 | 读取 StabilityConfig，缺失则使用默认配置并警告 |
| 执行规则 | 危险区下降、安全区回复、数值 clamp、归零只发一次 |
| 下降规则 | 下降 = base_decay × area × equipment × item × delta |
| 回复规则 | 回复不能超过 max_stability |
| 失败规则 | 不允许负数稳定值继续参与普通逻辑 |
| 边界规则 | StabilityComponent 不处理 UI、遮罩、音频、结算 |

### AI/MCP 开工提示词

```text
请实现“稳定值核心组件”。
必须读取 Doc/04_稳定值与视野系统/01_稳定值核心组件.md 和本小节。
新增 StabilityComponent.gd 和 StabilityConfig 默认配置。
实现 current/max、危险区下降、安全区回复、区域/装备/物品倍率、clamp、stability_changed、stability_depleted。
禁止实现 UI、音频、黑暗遮罩、死亡演出和正式物品系统。
验证：离开安全区下降，进入家回复，进入前哨回复，归零只触发一次 depleted。
最终报告新增文件、字段、信号、验证结果。
```

| 分类 | 规则 |

禁止实现遮罩绘制、UI、音频。

## 4. 视野半径控制器

对应文档：`03_视野半径控制器.md`

### 功能规则细化

| 分类 | 规则 |
|---|---|
| 触发条件 | stability_changed、safe_area_entered、safe_area_exited、depleted |
| 执行规则 | target 来自曲线，actual 平滑靠近 target |
| 最低规则 | 普通探索最低 3，死亡例外 |
| 前哨规则 | 已修复前哨目标半径 20 |
| 边界规则 | VisionController 不绘制遮罩，只传半径 |

### AI/MCP 开工提示词

```text
请实现“视野半径控制器”。
必须读取 Doc/04_稳定值与视野系统/03_视野半径控制器.md 和本小节。
新增 VisionController，监听稳定值变化，根据 StabilityConfig 计算 target_vision_radius，并用 0.15-0.3 秒平滑 actual_vision_radius。
actual 半径传给 DarknessMask；缺失时输出警告。
禁止实现 Shader、UI、音频和死亡结算。
验证 100/60/35/15/5 稳定值时半径变化和最低 3 规则。
```

## 13. 执行代理最终报告格式

任何模块完成后，执行代理必须汇报：

```text
完成模块：

新增/修改文件：
- ...

实现内容：
- ...

关键规则：
- ...

验证：
- 已运行/已手动验证：...
- 结果：...

未实现范围：
- ...

风险/后续：
- ...
```

如果无法验证，必须写：

```text
未验证：
- 原因：...
- 建议下一步验证方式：...
```
