# 09_物品装备与区域修正规则

> 对应母版章节：11、12、15  
> 施工补充：开工前必须读取 `13_模块功能规则细化与AI开工提示词.md` 的“物品装备与区域修正规则”小节。

---

## 1. 实现目标

为稳定值系统预留消耗品、装备词条和污染区域修正接口。V0.1 可以只做 C/B 回复物品或纯接口，但核心公式必须支持区域倍率、装备倍率和物品倍率。

---

## 2. 设计依据

回复性物品：

| 物品 | 品质 | 效果 |
|---|---|---|
| 破旧稳定剂 | C | 立即回复 15 稳定值 |
| 标准稳定剂 | B | 立即回复 25 稳定值 |
| 高纯稳定剂 | A | 立即回复 40 稳定值 |
| 异常稳定核心 | S | 立即回复至满，并短时减缓下降 |

停损类物品：

| 物品 | 品质 | 效果 |
|---|---|---|
| 镇定贴片 | B | 8秒内稳定值下降速度 -50% |
| 简易屏蔽器 | A | 6秒内稳定值不下降 |

污染倍率：

| 区域 | 稳定值下降倍率 |
|---|---:|
| 普通危险区 | 1.0 |
| 轻污染区 | 1.3 |
| 重污染区 | 1.8 |
| 异常核心区 | 2.5 |

---

## 3. 功能规则细化

| 分类 | 规则 |
|---|---|
| 回复物品 | 立即增加稳定值，不能超过 max |
| 停损物品 | 修改 item_decay_multiplier，有持续时间 |
| 装备修正 | 可影响 max、decay_rate、vision_radius、recovery_rate |
| 区域修正 | 修改 area_decay_multiplier，V0.1 只预留 |
| 失败规则 | 多个倍率来源必须可追踪，不能写成魔法数 |
| 反馈规则 | 调试面板显示 area/equipment/item multiplier |
| 边界规则 | 简易屏蔽器一类“停止下降”应通过倍率 0 表达，不能直接关掉系统 |

## 5. 数据结构

修正来源：

```gdscript
enum StabilityModifierSource {
    AREA,
    EQUIPMENT,
    ITEM,
    DEBUG
}
```

修正字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `source_id` | `StringName` | 修正来源 |
| `source_type` | `StabilityModifierSource` | 来源类型 |
| `decay_multiplier` | `float` | 下降倍率 |
| `vision_radius_bonus` | `float` | 视野加成 |
| `max_stability_bonus` | `float` | 最大值加成 |
| `recovery_rate_multiplier` | `float` | 回复倍率 |
| `duration` | `float` | -1 表示永久 |

---

## 6. 核心流程

使用回复物品：

```text
item_used
→ StabilityComponent.apply_instant_recovery(amount)
→ clamp 到 max
→ 发 stability_changed
```

使用停损物品：

```text
item_used
→ add_modifier(source_id, ITEM, decay_multiplier, duration)
→ 每帧更新 duration
→ 过期后移除 modifier
→ 重新计算 item_decay_multiplier
```

进入污染区：

```text
pollution_area_entered
→ set area_decay_multiplier
pollution_area_exited
→ reset area_decay_multiplier
```

---

## 7. 信号与接口

```gdscript
signal stability_modifier_added(source_id: StringName)
signal stability_modifier_removed(source_id: StringName)
signal stability_modifiers_recalculated(area: float, equipment: float, item: float)
```

接口：

```text
apply_instant_recovery(amount)
add_timed_decay_modifier(source_id, multiplier, duration)
set_area_decay_multiplier(multiplier)
set_equipment_decay_multiplier(multiplier)
```
