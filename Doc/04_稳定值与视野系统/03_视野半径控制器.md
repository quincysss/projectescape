# 03_视野半径控制器

> 对应母版章节：6、7.4、14.3  
> 施工补充：开工前必须读取 `13_模块功能规则细化与AI开工提示词.md` 的“视野半径控制器”小节。

---

## 1. 实现目标

实现 `VisionController`：监听稳定值变化，根据配置曲线计算目标视野半径，并用平滑方式更新实际视野半径，再传给 `DarknessMask`。它不负责稳定值数值，也不负责遮罩绘制细节。

---

## 2. 设计依据

基础视野数值：

| 阶段 | 稳定值范围 | 视野半径 |
|---|---:|---:|
| 稳定 | 100 - 60 | 20 - 18 |
| 轻度失稳 | 60 - 35 | 18 - 14 |
| 危险 | 35 - 15 | 14 - 8 |
| 濒临崩溃 | 15 - 0 | 8 - 3 |

规则：

```text
视野半径变化必须平滑。
普通探索状态最低视野不低于 3 单位。
稳定值为 0 时进入死亡黑幕收束，不再用普通视野规则。
前哨站内视野半径目标设为 20。
```

---

## 3. 功能规则细化

| 分类 | 规则 |
|---|---|
| 触发条件 | 稳定值变化、阶段变化、进入/离开前哨或家 |
| 执行规则 | 根据配置获取目标半径，实际半径用 lerp/tween 平滑靠近 |
| 平滑规则 | 平滑时间 `0.15 - 0.3` 秒 |
| 最低规则 | 非死亡探索状态实际半径不低于 3 |
| 前哨规则 | 已修复前哨内目标半径为 20，但仍为局部视野 |
| 失败规则 | DarknessMask 缺失时输出警告，但仍计算半径 |
| 反馈规则 | 调试面板显示 target/actual 半径 |

---

## 4. Godot 落地范围

| 路径 | 职责 |
|---|---|
| `res://scripts/vision/VisionController.gd` | 半径计算和平滑 |
| `res://scripts/vision/DarknessMask.gd` | 接收实际半径 |
| `res://scripts/debug/StabilityDebugPanel.gd` | 显示目标/实际半径 |

---

## 5. 数据结构

字段：

| 字段 | 类型 | 默认值 |
|---|---|---|
| `target_vision_radius` | `float` | `20` |
| `actual_vision_radius` | `float` | `20` |
| `minimum_exploration_radius` | `float` | `3` |
| `smooth_time` | `float` | `0.2` |
| `outpost_rest_radius` | `float` | `20` |
| `is_death_collapse_active` | `bool` | `false` |

---

## 6. 核心流程

```text
on_stability_changed(current)
→ 如果死亡收束中，返回
→ target = config.get_target_radius_for_value(current)
→ 如果 current > 0，target = max(target, minimum_exploration_radius)
→ 每帧 actual lerp 到 target
→ darkness_mask.set_vision_radius(actual)
```

前哨：

```text
进入已修复前哨
→ target_vision_radius = outpost_rest_radius
→ darkness_alpha 可交给 DarknessMask 变浅
```

---

## 7. 信号与接口

```gdscript
signal vision_radius_changed(actual_radius: float, target_radius: float)
signal vision_radius_target_changed(target_radius: float)
```

接收：

```text
StabilityComponent.stability_changed
StabilityComponent.stability_depleted
SafeArea2D.safe_area_entered/exited
```

## 9. 验收标准

| 编号 | 标准 |
|---|---|
| A1 | 稳定值变化会更新目标半径 |
| A2 | 实际半径平滑靠近目标半径 |
| A3 | 普通探索半径不低于 3 |
| A4 | 稳定值为 0 不走普通视野规则 |
| A5 | 已修复前哨内目标半径可恢复到 20 |
| A6 | 调试面板能显示 target/actual |
