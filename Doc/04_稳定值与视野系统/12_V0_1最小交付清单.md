## 1. 最小交付定义

V0.1 稳定值与视野系统最小交付只证明一件事：

```text
玩家离开安全区后稳定值持续下降，
视野随阶段收缩，
回到家或已修复前哨站后稳定值回复，
稳定值归零后进入死亡失败流程。
```

不是 V0.1 目标：

```text
完整精神污染系统
完整装备词条
复杂消耗品体系
正式音频混音
复杂 Shader 后处理
多人同步
```

## 5. 验收总清单

基础功能：

```text
[ ] 玩家在家中稳定值不下降。
[ ] 玩家离家后稳定值开始下降。
[ ] 玩家回家后稳定值快速回复。
[ ] 玩家进入未修复前哨站不会停止下降。
[ ] 玩家进入已修复前哨站会停止下降并回复。
[ ] 稳定值归零后触发死亡。
[ ] 死亡后进入失败结算接口。
```

视野：

```text
[ ] 稳定值 100 时视野舒适。
[ ] 稳定值 60 以上视野变化轻微。
[ ] 稳定值 35 以下明显压迫。
[ ] 稳定值 15 以下有强烈危机感。
[ ] 视野半径变化平滑。
[ ] 最低普通视野不低于 3 单位。
[ ] 死亡时黑幕收束正确。
```

节奏：

```text
[ ] 家 -> 中层普通容器 -> 开 2 秒 -> 回家，有轻微压力但有余量。
[ ] 家 -> 外层保险柜 -> 开 4.8 秒 -> 回家，无前哨时明显危险。
[ ] 有第一前哨后，外层探索压力可控。
[ ] 超重状态下从外层返回，负重和稳定值共同制造减重决策。
```
