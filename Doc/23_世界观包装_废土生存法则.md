## 1. 本文档一句话说明

《废土生存法则》的世界观不是“英雄清剿废土”，而是“一个幸存者从仍亮着灯的家中出发，在逐渐收窄的黑暗里，把废弃小镇一点点重新锚定为可行动空间”。

核心包装句：

```text
在被静潮吞没的废弃矿镇里，玩家躲在仍有锚灯的家中，借助残存信号规划路线，离开安全光域进入逐渐缩窄的黑暗，搜寻封存容器与前哨材料，修复临时锚站扩大行动半径，并在稳定崩溃前把能带回的东西送进局外仓库。
```

## 3. 世界观核心

## 3.1 世界状态

废弃小镇曾经是一座矿业与维修工业混合的小镇。小镇没有在一场爆炸中彻底毁灭，而是在一次被幸存者称为“静潮”的异常现象中逐渐失去可居住性。

“静潮”不是普通黑夜，也不是单纯毒雾。它更像一种会腐蚀感知、方向感、设备信号和身体稳定性的环境背景。

新剧情口径中，世界背景明确为：2050 年爆发史无前例的核战争，战后人类退入地下哨所生活。地表因核辐射和异常污染产生名为“暗潮”的物质，它无处不在，会吞噬暴露在地表的人类。早期文档中的“静潮”可视为幸存者早期称呼或局部现象名；对玩家主线叙事、对白和 UI 提示统一使用“暗潮”。

玩家来自 `404 哨所`，属于少数因辐射感染产生异常体质、能短暂抵御暗潮侵蚀的人。404 哨所能否继续生存、能否重返地面，依赖玩家持续探索地表、带回物资并修复地面前哨网络。

因此，玩家离开安全区后会发生：

```text
视野被黑暗压缩。
稳定值持续下降。
信号只能局部识别。
声音变闷、变急促。
返程判断越来越困难。
```

这能解释当前核心规则：

```text
黑暗遮罩 = 暗潮导致的视界收束。
稳定值 = 玩家在暗潮中保持行动、感知和信号同步的综合状态。
家和已修复前哨站 = 能暂时抵消暗潮影响的锚定区域。
稳定值归零死亡 = 玩家被暗潮完全吞没，行动能力和方向感崩溃。
```

---

## 3.2 家的意义

家不是豪华基地，而是一处被反复修补、仍然能维持锚灯的安全屋。

家承担三层意义：

```text
玩法层：出生、全图观察、稳定值回复、临时存储、撤离点。
叙事层：玩家仍然相信自己可以回去的地方。
美术层：在灰黑小镇中唯一长期明亮、有室内生活痕迹的建筑。
```

家中全图观察可以包装为：

```text
家里的旧监测台仍能读取小镇残存信号。
这些信号足够帮助玩家判断“哪里有机会”，但不足以看穿容器内部。
```

家不应被写成绝对安全的王国。它只是“目前还能撑住的地方”。这能保留废土压力，也能让每次回家有情绪回报。

---

## 3.3 前哨站的意义

前哨站是小镇旧维护网络留下的失效锚点。它们本来用于矿区应急、街区供电、通讯转发和短时避难，现在需要玩家用材料重新点亮。

前哨站包装原则：

```text
修复前：坏掉的锚点，能看到修复潜力，但不提供安全感。
修复中：玩家冒险把材料接入旧网络，稳定值仍在下降。
修复后：临时被点亮的安全支点，能恢复稳定值和存放少量物资。
```

前哨站不是第二个家。它没有完整全图观察，也不是永久设施。

叙事表达：

```text
家让玩家规划。
前哨让玩家走得更远。
```

---

## 3.4 容器的意义

容器不是“宝箱”，而是静潮后仍被封存、遗落或被旧系统锁住的物资单位。

容器提供的是“可能性”，不是承诺。

包装解释：

```text
家中只能读取容器回波，所以玩家知道那里有东西，但不知道里面是什么。
靠近后可以识别容器类型，因为玩家能看清外观和锁具。
打开后才知道产物，因为容器内部长期封存，信号无法穿透。
```

容器类型的叙事含义：

| 容器 | 包装定位 |
|---|---|
| 纸箱 | 被遗弃的普通补给，轻、快、低期待 |
| 木箱 | 老式运输箱，基础材料和杂物 |
| 工具柜 | 小镇维修网络遗留物，工具、材料、图纸倾向 |
| 医疗柜 | 旧诊所和救援点遗留物，稳定恢复物品倾向 |
| 小保险箱 | 私人或商铺封存物，中高期待 |
| 大保险柜 | 旧矿镇高价值封存物，高风险高期待 |
| 异常箱 | 被静潮污染或重写信号的容器，特殊但不保底 |

---

## 3.5 背包与负重的意义

负重不是单纯数值惩罚，而是废土叙事的身体感。

玩家每拿一件东西，都在背上一个问题：

```text
它值不值得占一格？
它重不重？
我能不能带着它回去？
要不要先放到家或前哨站？
如果死在外面，它会不会白拿？
```

包装表达：

```text
废土里不是“找到东西就拥有东西”。
只有带回、存住、入库，东西才真正属于玩家。
```

这条口径应贯穿背包、存储、结算和仓库文案。

---

## 3.6 撤离与仓库的意义

撤离不是逃离世界，而是把一次探索封存成长期资产。

局内物品归属可以包装为：

```text
背包：还在风险里。
家中存储：本局已安全封存。
前哨存储：本局临时封存。
结算池：本局可带回清单。
局外仓库：长期资产。
```

仓库的叙事价值：

```text
带回来的东西真的属于我了。
仓库空间有限，但处置权属于我。
系统不会静默吞掉我的收获。
```

## 4. 推荐世界名称与地点口径

V0.1 可以继续在正式 UI 中使用“废弃小镇”，但世界观包装建议预留一个可替换地名：

```text
灰井镇
```

命名依据：

```text
“灰”对应灰黑手绘废土视觉。
“井”对应矿业、地下设施、矿币、旧工业小镇。
两个字短，适合 UI、任务、物品描述和宣传语。
```

当前使用方式：

```text
正式文档中可写“废弃小镇 / 灰井镇，暂定”。
如果后续确定地名，再统一改 UI 与文案。
```

区域语义建议：

| 圈层 | 包装名称方向 | 叙事功能 |
|---|---|---|
| 家附近 / 内层 | 老居民区、旧门廊、街角铺面 | 教学、基础收益、第一次离家 |
| 中层 | 维修街、诊所街、仓储边道 | 容器和第一前哨推进 |
| 外层 | 矿务仓库、旧变电区、封锁街区 | 第二前哨、负重和返程压力 |
| 最外层 | 断讯区、灰井边缘、异常回波区 | 高期待容器、异常箱、后续事件 |

## 5. 玩家身份包装

玩家不是士兵、赏金猎人或救世主。

推荐身份：

```text
返镇拾荒者。
仍住在旧安全屋里的幸存者。
依靠背包、旧监测台、稳定装置和临时前哨一点点回收小镇。
```

角色气质：

```text
年轻。
瘦弱。
旧衣服。
旧背包。
少量紫色或蓝色电子模块。
不是重装军事风。
不是干净校园风。
```

玩家动机不需要一开始讲得很满。V0.1 可以只成立以下动机：

```text
家还需要材料。
仓库还缺东西。
小镇里还有能用的物资。
前哨站可以把更远的路重新点亮。
每次撤离都让下一次更有准备。
```

## 6. 叙事支柱

## 6.1 安全不是地点，是被维护出来的边界

家安全，是因为它还在工作。前哨安全，是因为玩家修好了它。

包装效果：

```text
玩家不是占领地图。
玩家是在临时修复地图。
```

---

## 6.2 黑暗不是敌人，是环境在收回世界

V0.1 没有战斗也能成立，因为主要敌人是：

```text
时间。
黑暗。
不确定收益。
负重。
返程距离。
自己的贪心。
```

文案应避免把黑暗写成具体怪物。它更像一种持续压迫的环境规律。

---

## 6.3 容器是诱惑，材料是承诺

普通容器代表未知收益，前哨材料代表主流程推进。

文案口径：

```text
容器问玩家：要不要赌？
材料问玩家：要不要推进？
前哨问玩家：要不要把这条路变安全一点？
```

---

## 6.4 带回才算拥有

拾取不是获得，入库才是拥有。

这句话应成为结算、仓库、丢弃、死亡文案的底层逻辑。

---

## 6.5 玩家每局都在重建一条路线

前哨站每局重置不是世界观漏洞，而是玩法主题：

```text
静潮会在每次探索后重新吞回临时锚点。
玩家带走的是物资、经验和局外成长，不是永久净化地图。
```

这能解释为什么前哨站本局有效，同时保留每局推进感。

## 8. 文案语气规则

## 8.1 总体语气

关键词：

```text
克制。
短句。
冷静。
有生活痕迹。
少解释，多暗示。
让系统信息清楚，不装神秘。
```

避免：

```text
中二宣告。
救世主使命。
军事命令腔。
长篇剧情旁白。
恐怖片式夸张威胁。
把每个 UI 都写成诗。
```

正确方向：

```text
灯还亮着。
外面在变窄。
这个箱子有回波。
你还能再开一个，但不一定能带回来。
```

## 9. 宣传与商店页包装

## 9.1 一句话卖点

```text
在被黑暗吞没的废弃矿镇里，从安全屋出发，修复临时前哨，带着有限的稳定值和背包容量，把还能用的东西带回家。
```

```text
《废土生存法则》是一款 2D 俯视角废土搜集撤离游戏。你从安全屋中观察废弃小镇的残存信号，离家后视野会随稳定值下降而不断收缩。开启未知容器、搜集前哨材料、修复临时安全点、管理负重，并在贪心把你拖进黑暗前完成撤离。带回来的物资将进入局外仓库，支撑下一次更远的探索。
```

## 9.3 标签语

```text
带回才算拥有。
灯亮着的地方，才叫安全。

前哨不是据点，是你争来的喘息。
再开一个箱子，还是现在回家？
```

## 10. 物品命名方向

## 10.1 普通材料

命名原则：

```text
旧工业。
可被研究或制作复用。
短、具体、能想象重量。
```

示例：

```text
废铁片
裂纹螺母
绝缘胶带
旧铜线
磨损轴承
滤芯残片
矿灯玻璃
```

## 10.2 前哨材料

命名原则：

```text
看起来能修设备。
比普通材料更有目标性。
不写成魔法材料。
```

示例：

```text
旧电池
稳定线圈
强化金属板
锚灯灯芯
手摇供电器
信号继电片
```

## 10.3 稳定恢复物品

命名原则：

```text
民用、简陋、带一点幸存者手工感。
不要像高级科幻药剂。
```

示例：

```text
安定糖
破旧稳定剂
镇定贴片
清醒饮料
简易屏蔽器
```

## 10.4 稀有物资与收藏物

命名原则：

```text
让玩家感觉小镇曾经有人生活。
不必都强功能化。
可以成为情绪奖励、收藏目标和高级制造价值来源。
```

示例：

```text
旧矿徽
未寄出的录音带
裂开的合照
封存账本
停摆怀表
矿务站钥匙牌
```

## 10.5 装备与耳机

命名原则：

```text
旧物改造。
功能清晰。
与负重、信号、稳定值相关。
```

示例：

```text
加固旧背包
小型工具背架
屏蔽耳机
矿工耳机
手工信号夹
旧式滤噪模块
```

## 11. 美术叙事口径

美术方向必须继续遵守 17 文档：

```text
灰黑手绘漫画线稿。
脏旧废土城市。
少量紫、蓝、红、暖黄点缀。
俯视或轻微斜俯视。
家和修复后前哨站明显更明亮。
修复前前哨站灰暗破损，不能提前像安全点。
```

颜色叙事建议：

| 颜色 | 叙事功能 |
|---|---|
| 灰黑 | 小镇主体、静潮压迫、废弃建筑 |
| 暖黄 | 家、修复后前哨、可喘息的灯 |
| 蓝色 | 旧设备、信号、监测台、耳机模块 |
| 紫色 | 异常、噪点、高价值但不稳定的机会 |
| 红粉 | 稳定值危险、低稳定警告、医疗或安定提示 |
| 绿色 | 少量植物、医疗补给，不宜大面积使用 |

场景叙事原则：

```text
普通建筑告诉玩家“这里已经坏了”。
容器告诉玩家“这里可能还有东西”。
前哨告诉玩家“这里可以被修好”。
家告诉玩家“这里仍然有人生活”。
```

---

## 12. V0.1 单局叙事节奏

V0.1 不需要复杂剧情，也能形成完整叙事：

```text
醒来，家里的灯还亮着。
查看旧监测台，确认小镇里的回波。
离开家，黑暗压过来，稳定值开始下降。
在有限视野里确认容器和材料。
开容器，赌一次未知收益。
捡前哨材料，背包开始变重。
回家或回前哨，存下不想丢的东西。
冒险修复前哨，把更远的路点亮。
修复第二前哨后，决定继续贪还是回家撤离。
撤离成功或探索失败。
结算，确认哪些东西真正被带回。
入库，准备下一次离家。
```

## 13. 后续扩展口径

## 13.1 战斗扩展

未来如果加入敌人，不建议立刻把世界观变成怪物清剿。

敌人方向可以是：

```text
被静潮影响的迷失者。
自动巡逻的旧维护设备。
争夺物资的其他幸存者。
异常区域中的临时威胁。
```

战斗应服务探索压力，而不是取代稳定值、负重和撤离作为主循环。

## 13.2 多人扩展

多人不是必须解释成大型阵营战争。

可扩展方向：

```text
多人共用一次灰井镇回收行动。
各自携带不同设备和背包。
共享或竞争前哨站。
容器开启期间可以互相保护或抢夺。
结算按个人、队伍或阵营归属拆分。
```

## 13.3 研究所与制作所扩展

研究和制作应包装为：

```text
把从小镇带回来的旧物重新理解、修补和改造。
```

不要写成高科技实验室突然制造未来装备。项目的质感应保持“旧物改造”和“幸存者手工维修”。

---

## 14. 禁止事项

世界观包装中禁止：

```text
把 V0.1 写成打怪主导。
把玩家写成军方精英或救世主。
把稳定值写成普通生命值。
把前哨站写成永久净化设施。
把容器写成必出奖励的宝箱。
把家写成无限功能基地。
把仓库满溢写成系统自动裁决玩家收获。
使用大量难懂专有名词遮蔽玩法信息。
用长剧情旁白打断核心闭环。
让文案暗示 V0.1 存在实际未实现的系统。
```

## 15. 使用检查清单

新增 UI 文案、物品名、地图区名、宣传语、资产提示词时，检查：

```text
是否仍围绕安全屋、黑暗、稳定值、容器、前哨、负重、撤离、仓库？
是否清楚区分“拾取、存储、结算、入库”的归属层级？
是否避免把容器奖励写死为确定收益？
是否避免把前哨站写成永久据点？
是否保留家作为唯一完整规划中心？
是否符合灰黑手绘废土与少量信号色的美术方向？
是否没有承诺当前版本未实现的战斗、敌人、剧情任务或多人功能？
```

## 16. 本文档结论

《废土生存法则》的世界观核心不是“废土有多大”，而是“安全有多小”。

玩家的每一局都应该像一次从灯下走进黑暗的回收行动：

```text
看见机会。
离开安全。
赌一次容器。
背起重量。
修好前哨。
决定是否贪心。
在黑暗合拢前回家。
把带回来的东西变成下一次离家的底气。
```

只要这条体验成立，后续战斗、装备、多人和局外成长都会有统一的世界观骨架。
