# 04_家安全区落地规则

> 对应母版章节：6、10.2、11.2  
> 施工补充：开工前必须读取 `12_模块功能规则细化与AI开工提示词.md` 的“家安全区落地规则”小节。

---

## 1. 实现目标

把“家”落实为局内唯一永久安全起点和撤离终点：有出生点、安全区触发范围、门口缓冲区、存储点、撤离点、全图观察入口和离家切换边界。

---

## 2. 设计依据

家提供：

```text
出生点
全图观察
稳定值快速回复
黑暗遮罩关闭
临时存储
路线规划
撤离按钮
```

家安全区建议：

| 项目 | 建议值 |
|---|---:|
| 家建筑视觉尺寸 | 10 × 8 单位 |
| 家安全区触发范围 | 12 × 10 单位 |
| 门口缓冲范围 | 向外延伸 2 - 3 单位 |
| 稳定值回复 | 10/秒 |

---

## 3. 功能规则细化

| 分类 | 规则 |
|---|---|
| 触发条件 | 玩家进入/离开 HomeArea 的 SafeArea2D |
| 前置条件 | HomeArea 默认激活，不需要修复 |
| 执行规则 | 进入家：停止稳定值下降、开始回复、遮罩淡出、镜头全图、显示信号 |
| 执行规则 | 离开家：镜头拉近、遮罩淡入、稳定值下降、全图信号关闭 |
| 缓冲规则 | 边界加入 0.2 - 0.3 秒状态缓冲，避免门口抖动反复切换 |
| 反馈规则 | 正式表现融入场景，调试模式可显示边界 |
| 边界规则 | 家不能只有一条出口；门口过渡区宽度 ≥3 单位 |
| 区块规则 | 家应放置在特殊安全区块上，安全区块连接至少 3 条街道，不用普通楼房轮廓围出门口道路 |

## 4. Godot 落地范围

| 路径 | 职责 |
|---|---|
| `res://scenes/maps/HomeArea.tscn` | 家区域 prefab，可选 |
| `res://scripts/map/HomeArea.gd` | 家安全区、出生点、撤离点、存储点引用 |
| `res://scripts/map/SafeArea2D.gd` | 通用安全区触发脚本 |
| `res://scenes/maps/TownMap.tscn` | 放置 HomeArea |

推荐节点：

```text
HomeArea
├── VisualRoot
├── HomeBlockArea
├── SafeArea2D
├── DoorBufferArea2D
├── PlayerSpawn
├── StoragePoint
└── ExtractPoint
```

---

## 5. 数据结构

`HomeArea` 字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `area_id` | `StringName` | 固定 `home` |
| `safe_area_size` | `Vector2` | 默认 `12 × 10` |
| `door_buffer_units` | `float` | `2 - 3` |
| `restore_rate` | `float` | `10` |
| `is_extraction_point` | `bool` | true |
| `supports_overview` | `bool` | true |
| `supports_storage` | `bool` | true |

---

## 6. 核心流程

```text
1. 在 TownMap 中放置 HomeArea。
2. HomeArea 中放 PlayerSpawn、StoragePoint、ExtractPoint。
3. 设置 SafeArea2D 覆盖家建筑和门口缓冲区。
4. 玩家进入 SafeArea2D 后发出 safe_area_entered("home")。
5. 玩家离开 SafeArea2D 后等待缓冲时间，再发出 safe_area_exited("home")。
6. 调试模式显示 SafeArea 和 DoorBuffer。
```

---

## 7. 信号与接口

`SafeArea2D` 发出：

```gdscript
signal safe_area_entered(area_id: StringName, area_type: StringName)
signal safe_area_exited(area_id: StringName, area_type: StringName)
```

提供给 02：

```text
RunDirector：进入/离开家状态切换
StabilityComponent：停止下降/快速回复
VisionController：关闭/开启黑暗遮罩
CameraController：HOME_OVERVIEW/PLAYER_FOLLOW
ExtractionController：检查撤离点
```

## 9. 验收标准

| 编号 | 标准 |
|---|---|
| A1 | 家区域有玩家出生点 |
| A2 | 家区域有存储点和撤离点 |
| A3 | 家安全区覆盖建筑和门口缓冲 |
| A4 | 进入家发出安全区进入事件 |
| A5 | 离开家发出安全区离开事件且有防抖 |
| A6 | 调试模式可显示安全区边界 |
| A7 | 家至少连接 3 条可选主路 |
