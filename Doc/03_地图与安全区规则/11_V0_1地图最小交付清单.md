## 1. 最小交付定义

V0.1 地图最小交付不是完整美术关卡，而是能支撑 02 核心循环跑通的可玩空间。

必须支持：

```text
玩家出生在家
→ 家中能观察主要路线和信号
→ 离家有多条路线
→ 能到达容器点和前哨材料点
→ 能修复第一前哨和第二前哨
→ 已修复前哨能作为临时安全点
→ 能返回家中撤离
→ 地图验证工具能检查关键问题
```

## 5. 验收总清单

通行：

```text
[ ] 玩家可以从家到达所有容器点位。
[ ] 玩家可以从家到达所有前哨候选点。
[ ] 玩家可以从第一前哨到达第二前哨候选区域。
[ ] 玩家不能走出地图边界。
[ ] 玩家不能进入区块。
[ ] 隐藏楼房视觉层后，道路拓扑仍然成立。
[ ] 玩家不会被建筑局部碰撞卡住。
```

时间：

```text
[ ] 家到内层普通点 5 - 10 秒。
[ ] 家到第一前哨候选点 10 - 18 秒。
[ ] 家到中层容器点 12 - 25 秒。
[ ] 家到第二前哨候选点 25 - 40 秒。
[ ] 第一前哨到第二前哨 15 - 30 秒。
[ ] 第二前哨到最外层高价值点 10 - 20 秒。
```

信号：

```text
[ ] 家中能看见容器信号，但看不出内部奖励。
[ ] 前哨材料信号比容器信号更明确。
[ ] 未修复前哨站需求面板可读。
[ ] 已修复前哨站状态明显。
[ ] 离家后视野外信号不会完整显示。
```

安全区：

```text
[ ] 家安全区包含门口缓冲。
[ ] 前哨安全区小于家。
[ ] 未修复前哨不停止稳定值下降。
[ ] 已修复前哨能停止稳定值下降并较慢回复。
```
