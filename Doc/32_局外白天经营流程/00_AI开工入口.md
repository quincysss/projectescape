## 3. 最新硬规则

```text
白天只分为两个可操作阶段：
1. 开店前准备阶段 DAY_PREP
2. 开店中营业阶段 SHOP_OPEN

SHOP_SETTLEMENT 是营业结束后的结算显示状态，不算新的可操作经营阶段。

DAY_PREP 不显示出发探索。
DAY_PREP 只能通过“开店营业”进入 SHOP_OPEN。
SHOP_OPEN 可因 60 秒倒计时结束进入 SHOP_SETTLEMENT。
SHOP_OPEN 也可因玩家主动点击“提前结束营业”进入 SHOP_SETTLEMENT。
SHOP_SETTLEMENT 关闭后进入 NIGHT。
NIGHT 才显示出发探索。
```

## 4. 禁止回退

- 不实现独立买卖页签。
- 不实现订单页签作为白天经营主界面。
- 不允许仓库原材料直接出售换矿币。
- 不允许 DAY_PREP 直接出发探索或直接 Loading。
- 不允许 SHOP_OPEN 中进行研究升级。
- 不允许 SHOP_OPEN 中通过制造所临时制作新货。
- 不允许空货台且仓库无可上架货物时强迫玩家等满 60 秒。

## 5. AI 开工提示词

```text
请实现 32 局外白天经营流程。

读取 29、30、32 目录和 09/10/31 相关接口文档后，完成：
1. outgame_phase 支持 DAY_PREP、SHOP_OPEN、SHOP_SETTLEMENT、NIGHT。
2. DAY_PREP 显示仓库、制造所、研究所、图鉴、需求榜、开店营业按钮，不显示出发探索。
3. SHOP_OPEN 显示货台、可上架 sale_good 列表、需求榜、营业倒计时、成交日志、提前结束营业按钮。
4. SHOP_OPEN 中可上架、补货、撤下、查看详情和等待自动成交。
5. 当货台为空且仓库没有可上架 sale_good 时，提前结束营业按钮高亮提示。
6. 玩家点击提前结束营业时，立刻冻结成交模拟并进入 SHOP_SETTLEMENT。
7. SHOP_SETTLEMENT 展示卖出物资、数量、单项收益和总收益；关闭后进入 NIGHT。
8. 所有扣货、回仓、加币必须走 ShelfInventoryService / WarehouseManager / CurrencyWallet。

不得实现旧独立买卖、订单页签、仓库直接出售或白天出发探索。
```
