# 07_负重背包与临时存储闭环

> 对应母版章节：5.6、8.8、10、12.3、17.3  
> 实现层级：物品携带约束、减重决策、家中临时存储。
> 施工补充：开工前必须读取 `12_模块功能规则细化与AI开工提示词.md` 的“负重背包与临时存储闭环”小节。

---

## 1. 实现目标

建立局内搜集的约束：玩家不能无限拾取。背包格子和负重共同限制容器奖励、前哨材料和消耗品。玩家回到家后可以把物资放入临时存储，降低负重，并在撤离成功后转入结算池。

---

## 2. 设计依据

负重影响：

```text
继续搜集能力
移动速度惩罚
返程风险
是否丢弃物品
是否回家存储
```

家中临时存储：

```text
V0.1 初始容量建议 4 格。
存入家中的物品，玩家死亡后不丢失。
撤离成功后，家中存储物品进入结算池。
本局结束后，家中存储清空。
```

## 4. 数据结构

物品字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `item_id` | `StringName` | 物品 ID |
| `display_name` | `String` | 名称 |
| `amount` | `int` | 数量 |
| `weight_per_unit` | `float` | 单位重量 |
| `stack_limit` | `int` | 堆叠上限 |
| `item_type` | `StringName` | 材料、容器奖励、消耗品等 |

负重阶段：

| 范围 | 阶段 | 建议表现 |
|---|---|---|
| 0-70% | `LIGHT` | 无惩罚 |
| 70-100% | `HEAVY` | 轻微减速 |
| >100% | `OVERLOADED` | 明显减速，可禁止继续拾取 |

V0.1 建议先使用：

```text
LIGHT：速度 100%
HEAVY：速度 85%
OVERLOADED：速度 60%，不可拾取会超重的物品
```

## 5. 核心流程

拾取检查：

```text
1. 容器或材料请求添加物品。
2. InventoryComponent 检查是否有可堆叠位置或空格。
3. WeightComponent 检查添加后是否超过允许上限。
4. 两项都通过才添加。
5. 添加成功后重新计算当前负重和阶段。
6. 阶段变化时通知移动组件和 UI。
```

家中存储：

```text
1. 玩家进入 HomeSafeZone。
2. HomeStorageUI 可打开。
3. 玩家从背包选择物品存入家中。
4. HomeStorageComponent 检查 4 格容量。
5. 存入成功后从背包移除。
6. 当前负重重新计算，速度惩罚刷新。
7. 撤离成功时，背包 + 家中存储进入结算池。
8. 本局结束后清空家中临时存储。
```

丢弃减重：

```text
V0.1 可以先做“从背包删除物品”的调试版。
正式版再接世界掉落物。
```

## 6. 信号与接口

`InventoryComponent` 发出：

```gdscript
signal inventory_changed(items: Array)
signal item_add_failed(item_id: StringName, reason: String)
```

`WeightComponent` 发出：

```gdscript
signal weight_changed(current_weight: float, max_weight: float, stage: int)
signal weight_stage_changed(old_stage: int, new_stage: int)
```

`HomeStorageComponent` 发出：

```gdscript
signal home_storage_changed(items: Array)
signal home_storage_full()
```

依赖接口：

```text
PlayerMovement.set_weight_speed_multiplier(value)
LootContainer 请求 InventoryComponent.add_item
OutpostMaterialPickup 请求 InventoryComponent.add_item
SettlementController 读取背包和家中存储
```

## 8. 验收标准

| 编号 | 标准 |
|---|---|
| A1 | 物品添加会检查背包格子 |
| A2 | 物品添加会检查负重上限 |
| A3 | 负重跨阶段时移动速度倍率变化 |
| A4 | 家中存储默认 4 格，满格后不可继续存入 |
| A5 | 存入家中后背包移除该物品，当前负重下降 |
| A6 | 死亡时家中存储不丢失，背包按死亡规则处理 |
| A7 | 撤离成功时背包和家中存储进入结算池 |
