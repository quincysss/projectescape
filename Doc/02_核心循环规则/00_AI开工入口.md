# 02 核心循环规则 AI 开工入口

> 本文件只负责 AI 读取路由。完整规则以母版和拆分文档为准。

## 1. 适用任务

单局状态机、开始初始化、家安全区、稳定值/容器/前哨/背包/撤离之间的主循环、局外天数推进。

## 2. 最短读取路径

1. `Doc/00_AI开工入口.md`
2. `Doc/00_AI文档读取规则.md`
3. `Doc/02_核心循环规则_修订版_废土生存法则.md`
4. `Doc/02_核心循环规则/00_拆分规范与落地文档写法.md`
5. 按任务读取下表中的具体文档。

| 任务 | 加读文档 |
|---|---|
| 单局流程/状态机 | `01_单局流程状态机.md`、`02_单局开始与初始化.md` |
| 家安全区/离家切换 | `03_家安全区观察与离家切换.md` |
| 跨系统闭环 | `04` 至 `10` 对应闭环文档 |
| 局外天数 | `13_局外天数推进规则.md` |
| 实现排期/总验收 | `11_V0_1最小可玩闭环任务清单.md`、`12_模块功能规则细化与AI开工提示词.md` |

## 3. 验收入口

优先使用 `11_V0_1最小可玩闭环任务清单.md`，具体功能再回到对应闭环文档的验收标准。

## 4. 同步要求

核心循环改动通常会影响多个系统。修改后检查 `03` 至 `10` 的系统入口是否需要同步。
