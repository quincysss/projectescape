# 03_家安全区观察与离家切换

> 对应母版章节：5、6、12.1  
> 实现层级：安全区状态、观察模式、离家切换。
> 施工补充：开工前必须读取 `12_模块功能规则细化与AI开工提示词.md` 的“家安全区观察与离家切换”小节。

---

## 1. 实现目标

让“家”成为玩家每局的初始安全点、观察点、回收点和撤离终点。玩家在家中可以观察信号、恢复稳定值、整理物品；离开家后进入危险区表现：镜头拉近、黑暗遮罩开启、稳定值开始下降。

---

## 2. 设计依据

家提供：

```text
全图观察
稳定值快速回复
物资临时存储
撤离按钮激活点
路线规划空间
局内节奏重置点
容器信号观察
前哨材料信号观察
```

离家触发条件：

```text
玩家从家安全区范围内移动到范围外。
```

离家后：

```text
镜头从全图观察切到玩家中心模式
黑暗遮罩开启
稳定值开始持续下降
全图观察关闭
玩家只能看到当前视野范围内地图
```

## 4. 数据结构

`SafeZone` 字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `zone_id` | `StringName` | 例如 `home` |
| `zone_type` | `StringName` | `home` 或 `outpost` |
| `is_active` | `bool` | 家默认激活，前哨站修复后激活 |
| `restore_stability` | `bool` | 是否恢复稳定值 |
| `allow_storage` | `bool` | 是否允许存储 |
| `allow_extraction` | `bool` | 是否允许撤离 |

`RunCameraController` 模式：

```gdscript
enum CameraMode {
    OVERVIEW,
    PLAYER_FOLLOW
}
```

---

## 5. 核心流程

进入家：

```text
1. Player 进入 HomeSafeZone。
2. SafeZone 发出 safe_zone_entered("home")。
3. 状态机进入 OBSERVE 或 RECOVER。
4. StabilityComponent 停止衰减并开始回复。
5. VisionController 关闭黑暗遮罩。
6. CameraController 切到 OVERVIEW。
7. UI 显示家中存储、容器信号、前哨材料信号。
```

离开家：

```text
1. Player 离开 HomeSafeZone。
2. SafeZone 发出 safe_zone_exited("home")。
3. 状态机进入 LEAVE_HOME。
4. CameraController 用 0.5 秒左右切到 PLAYER_FOLLOW。
5. VisionController 开启黑暗遮罩。
6. StabilityComponent 开始衰减。
7. 镜头与遮罩完成后状态机进入 SCAVENGE。
```

回家：

```text
1. Player 从危险区进入 HomeSafeZone。
2. 稳定值恢复速度为每秒 10 点，最高 100。
3. 黑暗遮罩淡出 0.3-0.5 秒。
4. 镜头拉远 0.6 秒。
5. 如果撤离已解锁，显示撤离按钮可交互。
```

---

## 6. 信号与接口

`SafeZone` 发出：

```gdscript
signal safe_zone_entered(zone_id: StringName, zone_type: StringName)
signal safe_zone_exited(zone_id: StringName, zone_type: StringName)
```

`HomeSafeZone` 提供：

```text
can_show_overview_signals() -> true
can_store_items() -> true
can_extract() -> 取决于撤离是否解锁
```

相邻系统响应：

```text
StabilityComponent：暂停/恢复衰减，开始/停止回复
RunCameraController：切换 OVERVIEW/PLAYER_FOLLOW
VisionController：开关黑暗遮罩
SignalDisplay：显示或隐藏容器/材料信号
ExtractionController：检查撤离按钮可用性
```

## 8. 验收标准

| 编号 | 标准 |
|---|---|
| A1 | 玩家出生时处于家安全区，状态机进入 `OBSERVE` |
| A2 | 家中稳定值不下降，低于 100 时按每秒 10 点恢复 |
| A3 | 家中黑暗遮罩关闭，镜头为全图观察 |
| A4 | 离开家后镜头切到玩家跟随，黑暗遮罩开启 |
| A5 | 离家后稳定值开始下降 |
| A6 | 回家后黑暗遮罩淡出，镜头拉远，稳定值恢复 |
| A7 | 只有撤离解锁后，家中撤离按钮才可用 |
