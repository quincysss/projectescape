# 10-08 调试工具与验证清单

> 来源文档：`10_研究所与制作所系统_修订版_废土生存法则.md`  
> 目标：定义研究、制造所解锁、图纸和制作的 Debug 操作与验收方式。

## 1. Debug 面板

局外 BaseScene 可保留开发期 Debug 面板，但所有按钮必须调用正式接口。

建议按钮：

```text
+500 矿币
加入可售卖道具
补齐研究资源
完成下一档研究
重置研究
激活第一章目标
补齐制造所解锁矿币
解锁制造所
添加指定图纸
学习指定图纸
添加制作材料
制作指定配方
清空制作/图纸存档
```

禁止：

```text
UI 直接改 currencies。
UI 直接改 warehouse_items。
UI 直接改 research_levels。
UI 直接改 manufacturing_station_unlocked。
UI 直接插入 unlocked_recipe_ids。
```

## 2. 研究验证

必须验证：

```text
材料不足时不能研究。
矿币不足时不能研究。
研究失败不扣材料、不扣矿币。
研究确认前取消不扣资源。
研究确认后扣除材料和矿币。
研究等级写入 research_levels。
研究效果下一局或立即按设计生效。
重启后研究等级保留。
```

容量研究重点：

```text
未研究局内背包为 8 格。
inventory_slots Lv.3 后局内背包为 20 格。
未研究家中安全箱为 1 格。
home_storage_slots Lv.3 后家中安全箱为 4 格。
未研究前哨安全箱为 0 格，不自动打开前哨存储。
outpost_storage_slots Lv.2 后前哨安全箱为 2 格。
max_stability Lv.5 后稳定值上限为 200。
warehouse_capacity 按 80 -> 120 口径生效。
```

仓库容量重点：

```text
未研究：80。
Lv.1：80。
Lv.2：90。
Lv.3：100。
Lv.4：110。
Lv.5：120。
```

## 3. 制造所解锁验证

必须验证：

```text
新档制造所未解锁。
首次成功返回剧情前，制造所不能解锁。
第一章目标激活后，4999 矿币不能解锁。
5000 矿币可以解锁。
解锁成功后扣除 5000 mine_coin。
解锁成功后 manufacturing_station_unlocked=true。
解锁成功后 chapter_1_completed=true。
重启后制造所仍可用。
保存失败时回滚扣币和 flag。
```

## 4. 图纸验证

实现图纸后必须验证：

```text
仓库没有图纸时不能学习。
已解锁配方不能重复学习同一图纸。
学习图纸会消耗 1 件图纸物品。
学习后写入 learned_blueprint_ids。
学习后写入 unlocked_recipe_ids。
重启后配方仍解锁。
学习失败不扣图纸。
重复图纸不会自动删除或分解。
```

## 5. 制作验证

实现制作后必须验证：

```text
制造所未解锁时不能制作。
未解锁配方不能制作。
材料不足时不能制作。
矿币不足时不能制作。
仓库空间不足时不能制作，或产物进入待领取列表。
确认前取消不扣材料。
确认后扣除材料并生成产物。
产物进入局外仓库并保存。
重启后产物仍在仓库中。
制作失败不吞材料、不吞产物。
```

## 6. Tooltip 验证

```text
研究材料图标 hover 显示局外道具详情。
图纸图标 hover 显示局外道具详情。
配方产物图标 hover 显示局外道具详情。
配方材料图标 hover 显示局外道具详情。
Tooltip 不触发扣除、学习、制作或保存。
切换页签、滚动、弹窗时 Tooltip 隐藏。
```
