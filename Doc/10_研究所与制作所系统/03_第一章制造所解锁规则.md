# 10-03 第一章高级制造所解锁规则

> 来源文档：`10_研究所与制作所系统_修订版_废土生存法则.md`
> 关联文档：`25_主菜单玩家档案与剧情章节目标系统/04_制造所解锁与第一章结束.md`

## 1. 实现目标

基础手作台是新档默认可用功能，负责把局内带回的低级材料加工为 C 级可售物资，保证白天开店闭环成立。

第一章目标是通过地面探索、制造可售物资和白天开店成交积攒矿币，花费 5000 矿币解锁高级制造所。高级制造所解锁后，开放更高等级可售物资、装备、消耗品和后勤订购标准件相关功能。

文案层可称为“制造所”，系统模块可沿用已有“制作所”命名。

## 2. 解锁条件

```text
current_chapter == 1
chapter_1_goal_active == true
advanced_manufacturing_station_unlocked == false
mine_coin >= 5000
```

费用：

```text
5000 mine_coin / 5000 矿币
```

## 3. 白天准备入口状态

白天准备阶段功能入口：

```text
仓库 / 制造所 / 研究所 / 图鉴
```

高级制造所未解锁时：

- 制造所入口可点击。
- 基础手作台分区可用。
- 高级制造所分区带锁。
- 点击后进入锁定说明页，而不是空白页。
- 显示矿币进度 `current / 5000`。
- 矿币不足时按钮置灰。
- 矿币足够时显示 `解锁高级制造所`。

制造所解锁页如果展示制作材料、产物预览或图纸图标，必须接入 `Doc/09_局外仓库系统/18_局外道具详情Tooltip.md` 的局外 Tooltip；单纯的矿币进度不使用道具 Tooltip。

## 4. 解锁后

高级制造所解锁成功后：

- 扣除 5000 矿币。
- `advanced_manufacturing_station_unlocked = true`
- `chapter_1_completed = true`
- 高级制造所分区变为正常可用。
- 弹出第一章结束弹窗。

## 5. 原子性

解锁必须通过服务处理：

```text
ManufacturingUnlockService
-> CurrencyWallet.spend("mine_coin", 5000)
-> ProfileService.set_feature_unlocked("advanced_manufacturing_station")
-> SaveService.save_all()
```

禁止 UI 直接扣 currencies 或直接修改制作所解锁 flag。

## 6. AI/MCP 开工提示词

```text
请实现第一章高级制造所解锁规则。

要求：
1. 基础手作台新档默认可用，只开放 C 级可售物资配方。
2. 高级制造所新档默认锁定。
3. 锁定页显示“需要矿币 current / 5000”。
4. 矿币不足时解锁按钮置灰；矿币足够时可点击。
5. 点击后显示确认弹窗，确认后通过 CurrencyWallet 扣除 5000 mine_coin。
6. 扣币成功后设置 advanced_manufacturing_station_unlocked=true、chapter_1_completed=true。
7. 解锁成功后高级制造所分区可用，并触发 25-04 的第一章结束弹窗。
8. 解锁失败不得扣币，不得改变 flag。
9. 不实现旧独立交易入口，不允许仓库原材料直接换钱。
```

## 7. 验收标准

- 新档基础手作台可用。
- 新档高级制造所锁定。
- 4999 矿币不能解锁。
- 5000 矿币可以解锁。
- 解锁后扣币并保存。
- 重启后高级制造所仍可用。
