# 10-03 第一章制造所解锁规则

> 来源文档：`10_研究所与制作所系统_修订版_废土生存法则.md`
> 关联文档：`25_主菜单玩家档案与剧情章节目标系统/04_制造所解锁与第一章结束.md`

## 1. 实现目标

制作所/制造所不是新档默认可用功能。第一章目标是通过地面探索和商人出售积攒矿币，花费 5000 矿币解锁制造所。

文案层可称为“制造所”，系统模块可沿用已有“制作所”命名。

## 2. 解锁条件

```text
current_chapter == 1
chapter_1_goal_active == true
manufacturing_station_unlocked == false
mine_coin >= 5000
```

费用：

```text
5000 mine_coin / 5000 矿币
```

## 3. 页签状态

局外顶部页签：

```text
仓库 / 商人 / 研究所 / 制造所
```

制造所未解锁时：

- 页签可显示，但带锁。
- 点击后进入锁定说明页，而不是空白页。
- 显示矿币进度 `current / 5000`。
- 矿币不足时按钮置灰。
- 矿币足够时显示 `解锁制造所`。

## 4. 解锁后

解锁成功后：

- 扣除 5000 矿币。
- `manufacturing_station_unlocked = true`
- `chapter_1_completed = true`
- 制造所页签变为正常可用。
- 弹出第一章结束弹窗。

## 5. 原子性

解锁必须通过服务处理：

```text
ManufacturingUnlockService
-> CurrencyWallet.spend("mine_coin", 5000)
-> ProfileService.set_feature_unlocked("manufacturing_station")
-> SaveService.save_all()
```

禁止 UI 直接扣 currencies 或直接修改制作所解锁 flag。

## 6. AI/MCP 开工提示词

```text
请实现第一章制造所解锁规则。

要求：
1. 制造所新档默认锁定。
2. 锁定页显示“需要矿币 current / 5000”。
3. 矿币不足时解锁按钮置灰；矿币足够时可点击。
4. 点击后显示确认弹窗，确认后通过 CurrencyWallet 扣除 5000 mine_coin。
5. 扣币成功后设置 manufacturing_station_unlocked=true、chapter_1_completed=true。
6. 解锁成功后制造所页签可用，并触发 25-04 的第一章结束弹窗。
7. 解锁失败不得扣币，不得改变 flag。
```

## 7. 验收标准

- 新档制造所锁定。
- 4999 矿币不能解锁。
- 5000 矿币可以解锁。
- 解锁后扣币并保存。
- 重启后制造所仍可用。
