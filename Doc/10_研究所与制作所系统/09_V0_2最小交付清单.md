## 1. 最小闭环

```text
玩家完成序章探索并返回局外
-> 无论撤离成功或失败，获得一次性开店周转物资
-> 物品进入局外仓库
-> 制造所制造 C 级可售物资
-> 白天开店，货台成交获得矿币
```

## 4. 完成定义

- 研究所能完成至少一条研究线，并影响下一局或局外容量。
- 仓库容量按 `80 -> 120` 当前口径生效。
- 制造所默认可用，且周转物资至少能制造 1 个 C 级可售物资。
- 第一章目标通过制造、上架和首次营业结算完成。
- 配方可用性由研究、章节、任务、店铺等级、设备、商人或地点事件条件决定。
- 至少一个配方能消耗材料并生成产物入库。
- 可售物资能交给 09 货台系统上架，成交获得矿币。
- 所有消耗失败都不吞材料、不吞货币。
- 研究、配方可用条件、配方、首次开店教学状态都能保存和重载。
- 局外道具图标 Tooltip 在研究和制作界面可用。
