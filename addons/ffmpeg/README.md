# EIRTeam.FFmpeg

This folder contains the EIRTeam.FFmpeg GDExtension used to load MP4/H.264 videos through Godot `VideoStreamPlayer`.

Source: https://github.com/EIRTeam/EIRTeam.FFmpeg

Project usage:
- Opening cinematic: `res://assets/cinematics/source/opening_intro_cinematic_720p.mp4`
- Main menu background loop: `res://assets/cinematics/main_menu/main_menu_background_loop_1080p.mp4`
